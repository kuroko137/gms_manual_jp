ルームマネージャー

アセットブラウザでは作成したルームを、プロジェクトを自分にとって使いやすく整理できるようフォルダグループに移すことができます。ただ GameMaker Studio 2 のゲームにおいて、ルームには決められた順番があり、開始ルームだけは指定しなくてはならず（ゲームはこのルームから開始され、それからコードやアクションで別のルームに移動できます）、そこで出番となるのがルームマネージャーです。

ルームマネージャーはアセットブラウザ上部の「追加メニュー」ボタンをクリックすると表示されるメニューから開けます。また、アセットブラウザのルームの左に表示されているアイコンをクリックしても開くことができます。

ルームマネージャーには 2 つのタブがあり、それぞれルーム順序、ルーム継承関係に分かれています。ルーム順序とは名前のとおり、room_goto_next()関数によってゲームを最初から最後まで辿っていくときに開けるルームの順番となります。先頭のルームには「ホーム」アイコンが表示され、ゲームの開始ルームとして指定されます。ホームアイコンはアセットブラウザ内でも表示されるため、アセットやフィルターの設定を問わず、どのルームが開始ルームなのかすぐに見分けることができます。

ルームの順番はリストのルームをドラッグして移動させるだけで変更できます。または / を押しながらクリックすると複数のルームを選択し、まとめて移動することも可能です。

ルーム継承関係のタブからは、「子」ルームの設定を行えます。子ルームとは「親」ルームをコピーしたもので、すべてのプロパティを引き継いでいます。何も手を加えなければ元のルームの単なるコピーですが、子ルームではルームエディタからレイヤー、インスタンスといった各要素の継承をそれぞれ切り替えたり、さらには新しいレイヤー、インスタンス、プロパティを追加するといったこともできます。

ルーム順序タブと同じようにドラッグしてルームの順番を変更できますが、子ルームを別の親ルームに移した場合は、元のルームの継承関係が破棄され、新しいルームのプロパティが引き継がれることになります。また、子ルームはリストの左側にドラッグすることで何も継承しないすべてを完備したルームに変換でき（リストの端が強調表示されます）、ドラッグした時点で独立したルームとなります。

ルームには子を複数追加したり、次のように子を親にして新しい子を追加することも可能です。

この例では親ルームとなる "rm_Level_1_Day" 、その子ルームとなる "rm_Level_1_Day_AfterBattle" 、"rm_Level_1_Night" があり、さらに "rm_Level_1_Night" には "rm_Level_1_Night_AfterBattle" という別の子ルームが割り当てられています。このように設定すると 1 つの親ルームのリソース、レベルデザインを再利用し、子ルームには必要最小限の変更を加えることができます。

メモ: ルームの継承についての詳細はこちらを参照してください。

最後に、ルームマネージャーには操作やルームの整理がしやすくなるツールがいくつかあります。上部にあるのは検索バーで、検索したい文字を入力すれば、その文字が含まれているルームに手早く移動することができます。一致したルームは強調表示され、検索バーに表示されている矢印ボタンをクリックすると別のルームに切り替わります。

ルーム順序のタブでは、下側に表示されているボタンをクリックしてルームの順番を上下に移動させることができます。

これらのボタンの横にはグループ作成ボタンがあり、クリックするとグループフォルダを作成できます。このフォルダはルームを整理しやすくするためのもので（ルームの順番は保たれます）、ルームを中にドラッグして追加します。または / を押しながらクリックすればルームを複数選択でき、そのままグループ作成ボタンをクリックすれば選択中のルームがすべてグループに追加されます。なお、グループに追加するとルームの順番は変更され、グループフォルダを削除した場合、中にあるルームはすべて削除されます（デフォルトでは警告が表示されます）。

また、ルームマネージャーではルームを右クリックすると以下のメニューが表示されます。

メニューには次の項目があります。

  • 子ルームを作成: クリックしたルームの子となり、親ルームのプロパティを引き継ぐ新しいルームを作成します。
  • 編集: ルームをルームエディタで開きます。
  • タグを編集: タグエディタを開き、ルームにタグを追加します。
  • 名前変更: ルームアセットの名前を変更します（ルームをゆっくりダブルクリックしても名前を変更できます）
  • 複製: 選択中のルームを複製し、現在のルームの下に追加します。
  • お気に入りに追加: 選択中のルームをアセットブラウザのお気に入り項目に追加します。
  • 削除: クリックすると選択中のルームが削除されます。グループフォルダを削除した場合、そのフォルダの中にあるルームもすべて削除されます。親ルームを削除した場合、削除されるのは親ルームだけとなり、子ルームは残ります。ただしすべて独立したルームとなり、子ルームではなくなります。
  • 子を展開: グループフォルダを選んでから実行すると、そのフォルダの中身を開くことができます。
  • すべて展開: すべてのフォルダグループを開きます。
  • すべて折りたたむ: すべてのフォルダグループを閉じます。
  • 整列: サブメニューが開き、ルームマネージャーのルームを上下に移動できます。ウィンドウ下部に表示されている各ボタンと同じものです。
  • エクスプローラーで開く: OS のファイルエクスプローラーから現在のプロジェクトのアセットを開きます。
  • アセットの色: ルームマネージャーで見やすくなるよう、ルームの色を指定できます。この色はアセットブラウザにも反映され、すでに設定されている色は上書きされます。
