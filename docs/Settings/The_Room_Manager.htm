ルームマネージャー

アセットブラウザではプロジェクトを使いやすく整理できるよう、作成したルームをフォルダに移すことができます。いっぽうで GameMaker Studio 2 のゲームはルームの順番が決められており、さらに開始ルームを指定しなくてはなりません（ゲーム開始後、コードやアクションによってこのルームから別のルームに移動できます）。そこで出番となるのがルームマネージャーです。

ルームマネージャーは、アセットブラウザ上部の「追加メニュー」ボタンをクリックして表示されるメニューから開けますが、アセットブラウザの各ルーム左側にあるアイコンをクリックしても開けます。

ルームマネージャーには 2 つのタブがあり、それぞれルーム順序、ルーム継承関係に分かれています。ルーム順序とは名前のとおり、room_goto_next()関数によってゲームを最初から最後まで辿っていくときに開けるルームの順番となります。先頭のルームには、ゲームの開始ルームであることを意味する「ホーム」アイコンが表示されます。このアイコンはアセットブラウザ内でも表示されるため、アセットやフィルターの設定を問わず、どのルームが開始ルームなのか簡単に見分けられるでしょう。

ルームの順番を変更するには、リストに表示されているルームをドラッグして移動させてください。または / を押しながらクリックすると複数のルームを選択し、まとめて移動することも可能です。

ルーム継承関係のタブからは「子」ルームの設定を行えます。子ルームとは「親」ルームをコピーしたもので、親からすべてのプロパティを引き継ぎます。そのままでは元のルームの単なるコピーですが、ルームエディタからレイヤー、インスタンスといった各要素の継承をそれぞれ切り替えたり、新しいレイヤー、インスタンス、プロパティを追加するといった変更を加えることが可能です。

ルーム順序タブと同じようにドラッグして順番を変更できますが、子ルームを別の親ルームに移した場合、元のルームとの継承関係は破棄され、別のルームのプロパティが引き継がれます。さらに子ルームはリストの左側にドラッグすると（リストの端が強調して表示されます）、何も継承しない独立したルームに変換できます。

この例では親ルームとなる "rm_Level_1_Day" 、その子ルームとなる "rm_Level_1_Day_AfterBattle" 、"rm_Level_1_Night" があり、そのうち "rm_Level_1_Night" には "rm_Level_1_Night_AfterBattle" という別の子ルームが割り当てられています。このようにすれば 1 つの親ルームのリソース、レベルデザインを再利用でき、子ルームには必要最低限の変更を加えるだけで済むようになります。

ルームマネージャーには他にも、ルームを整理したり切り替えるためのツールがあります。上部にあるのは検索バーで、検索した文字列が含まれているルームに手早く移動できます。見つかったルームは強調表示されるだけでなく、検索バーの矢印ボタンをクリックして隣のルームに切り替えることが可能です。

ルーム順序のタブでは、下側に表示されているボタンをクリックしてルームの順番を上下に移動できます。

一番奥にあるボタンはグループ作成ボタンで、クリックするとグループフォルダを作成できます。このフォルダはルームを整理しやすくするためのもので（ルームの順番は保たれます）、中にドラッグしてルームを追加できます。または / を押しながらクリックすればルームを複数選択でき、そのままグループ作成ボタンをクリックすれば選択中のルームがすべてグループに追加されます。なお、この操作をするとルームの順番は変更され、フォルダを削除した場合、中にあるルームはすべて削除されます（デフォルトでは警告が表示されます）。

また、ルームマネージャーではルームの右クリックメニューを開くことができます。

この右クリックメニューには次の項目があります。

  • 子ルームを作成: クリックしたルームのプロパティを引き継ぐ、新しい子ルームを作成します。
  • 編集: ルームをルームエディタで開きます。
  • タグを編集: タグエディタを開き、ルームにタグを追加します。
  • 名前変更: ルームアセットの名前を変更します（ルームをゆっくりダブルクリックしても変更できます）
  • 複製: 選択中のルームを複製し、現在のルームの下に追加します。
  • お気に入りに追加: 選択中のルームをアセットブラウザのお気に入り項目に追加します。
  • 削除: クリックすると選択中のルームが削除されます。グループフォルダを削除した場合、そのフォルダの中にあるルームもすべて削除されます。親ルームを削除した場合、削除されるのは親ルームだけとなり、子ルームは独立したルームとして残ります。
  • 子を展開: グループフォルダを選択してから実行すると、そのフォルダの中身を開くことができます。
  • すべて展開: すべてのグループフォルダを開きます。
  • すべて折りたたむ: すべてのグループフォルダを閉じます。
  • 整列: サブメニューが開き、ルームマネージャーのルームを上下に移動できます。ウィンドウ下部に表示されている各ボタンと同じ役割です。
  • エクスプローラーで開く: OS のファイルエクスプローラーから現在のプロジェクトのアセットを開きます。
  • アセットの色: ルームマネージャーで見やすくなるよう、ルームの色を指定します。この色はアセットブラウザにも反映され、すでに設定されている色は上書きされます。
