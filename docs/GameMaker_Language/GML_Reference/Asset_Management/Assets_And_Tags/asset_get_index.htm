asset_get_index

この関数を使用すると、ゲームアセットの名前から固有の識別インデックスを取得できます。アセットが見つからなかった場合は-1 の値が返され、見つかった場合は、チェックされたアセットの固有インデックス ID が返されます。この ID は他の関数において、たとえば sprite_index や path_index といった他のインデックス値と同じように扱うことができます。この関数では文字列からもアセットを参照できますが（以下の例を参照）、使用する前に必ずアセットが存在することを確認してください。存在しない場合はエラーが表示され、ゲームが強制終了する恐れがあります。

 

構文:

asset_get_index(name);

引数 説明
name インデックスを取得するゲームアセットの名前（文字列）です。

 

戻り値:

アセットのインデックス（整数）

 

例:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
    {
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
    }

上記のコードは文字列からオブジェクトのインデックスを取得し、インデックスが存在する場合、ゲーム内にオブジェクトのインスタンスを生成します。
