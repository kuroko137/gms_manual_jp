audio_set_master_gain

この関数を使用すると、指定したリスナーのすべてのサウンド・BGM に適用されるグローバル音量の絶対値を設定することができます。リスナーのインデックス初期値は 0 ですが、audio_get_listener_info 関数を利用するとターゲットプラットフォームで使用可能な各インデックスを取得できます。ゲインの値は 0 （無音）から 1 （最大音量）までのリニアスケールに基づいており、ゲーム中にリスナーを通して再生されるすべてのサウンド・BGM の相対ボリュームに影響します。

 

構文:

audio_set_master_gain(listenerIndex, gain);

引数 説明
listenerIndex ゲインを設定するリスナーのインデックスです。
gain グローバル音量の値 （0 から 1）です。

 

戻り値:

なし

 

例:

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
   {
   var info = audio_get_listener_info(i);
   audio_set_master_gain(info[? "index"], 0.75);
   ds_map_destroy(info);
   }

上記のコードは有効なすべてのリスナーをループで取得し、マスターゲインを 0.75 に設定しています。
