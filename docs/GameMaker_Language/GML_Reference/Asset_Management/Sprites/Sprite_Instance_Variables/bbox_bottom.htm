bbox_bottom

この読み込み専用変数は、インスタンスの境界ボックスの底の（ルーム内の）y 座標を返します。境界ボックスは、インスタンスのマスクの最大幅と最大高さ（ sprite_index または mask_index で設定）で定義されます。スプライトが正確なコリジョンマスクを持っている場合でも、境界ボックスは存在していて何かしらに使われているので、この変数を使って見つけることができます。インスタンスにスプライトが割り当てられていない場合は、返される値はインスタンスの y 座標と同じになることに注意してください。

 

構文:

bbox_bottom;

 

戻り値:

Real (integer)

 

例:

if bbox_bottom > room_height
    {
    y = room_height - sprite_yoffset;
    }

上記のコードは、インスタンスの境界ボックスがルーム外にあるかどうかをチェックし、外にある場合はインスタンスを新しい位置に設定します。
