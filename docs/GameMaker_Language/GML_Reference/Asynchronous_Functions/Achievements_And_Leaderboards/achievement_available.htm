achievement_available

この関数はユーザーが現在インターネットに接続されており、選択されたリーダーボードまたは実績が利用可能な場合に true を、そうでない場合は false を返します。

 

構文:

achievement_available()

 

戻り値:

Boolean

 

例:

if achievement_available()
    {
    achievement_post("ach_Level1", 100);
    }

上記のコードはリーダーボードのサービスが利用可能かどうかをチェックし、利用可能な場合は投稿します。
