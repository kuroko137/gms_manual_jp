variable_struct_remove

With this function you can remove a variable from a struct. You supply the truct reference to remove the variable from, and the name of the variable to remove, as a string.

 

Syntax:

variable_struct_set(struct, name);

Argument Description
struct The struct reference to remove the variable from
name The name of the variable to remove (as a string)

 

Returns:

N/A

 

Example:

if variable_struct_exists(mystruct, "shields")
   {
   variable_struct_remove(mystruct, "shields");
   }

The above code will check to see if the given variable exists in the given struct and if it does then it is removed.
