Steam

このセクションは Steam プラットフォームにゲームを公開できる、Steam API へのアクセス権が与えられているユーザーが対象となります。これらの関数を利用するには、事前にパブリッシャーまたはセルフパブリッシュシステムを介して Steam に承認されている必要があり、なおかつ Steam 環境設定および Steam ゲームオプションを正しく設定しなければなりません。

Steam API にまつわる関数は非常に多く用意されているため、閲覧しやすくなるよう以下のセクションに分けられています。

 

  • 一般 Steam API
  • Steam オーバーレイ
  • リーダーボード
  • 実績と統計
  • Steam Cloud
  • DLC (ダウンロードコンテンツ)
  • UGC (ユーザー生成コンテンツ)
