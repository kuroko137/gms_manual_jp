キーボードショートカット

GameMaker Studio 2 では非常に豊富なキーボードショートカットを利用できます。そのほとんどは IDE の各メニュー項目に表示されていますが、中には最初から示されていないショートカットもあります。このページではすべてのキーボードショートカットが一覧となっており、簡単に調べることができます。ショートカットはおおむね次の 3 つのカテゴリに分別されます。

  • グローバル - グローバルショートカットは IDE のどこからでも利用できるショートカットです。特定のウィンドウ、ガジェットにフォーカスされている場合はそちらのショートカットが優先されます。
  • ウィンドウ - ウィンドウショートカットは単一のウィンドウ・タブ用のショートカットです。ガジェットショートカットが優先される場合があります。
  • ガジェット - ガジェットショートカットは限定的なショートカットで、単一のガジェット（カラーピッカーなど）でのみ使用できます。

以下はエディタごとに段落分けされたショートカット一覧です。
