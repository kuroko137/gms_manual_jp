タイムラインエディタ

多くのゲームでは、決められた瞬間にアクションを起こしたい場面があります。Alarm イベントを利用すればそのようなことも実現できますが、処理が複雑になると上手くいきません。なにより Alarm は 12 個までしか利用できないのです。そんなときのために用意されているのがタイムラインアセットです。タイムラインでは、ゲーム時間の決められた瞬間（モーメント）に起こすアクションを指定することができ、各アクションはオブジェクトのイベントやコードから定義して利用できます。作成したタイムラインはオブジェクトへの割り当てが可能で、インスタンスがルームに配置または生成されると、指定したタイミングでアクションを順番に実行します。

たとえば、20 ステップかけて左に移動し、10 ステップかけて上に、20 ステップかけて右に、10 ステップかけて下に移動してから停止する見張りのオブジェクトを作りたいとしましょう。このような処理を行うには、タイムラインを作成してまずは左に動くように設定します。次にモーメントを 20 で作成し（モーメント（瞬間）とは、ゲームステップやゲームフレームを言い換えた用語です）、上に動くよう設定します。そしてモーメントを 30 で作成して右に、50 で作成して下に動くよう設定し、最後に 60 で動きを止めれば完成です。すべて終わったら、見張りのオブジェクトにタイムラインを割り当て、インスタンスをルームの置きたい場所に配置しましょう。タイムラインでは Alarm を設定し、一定時間過ぎたら処理を繰り返すこともできるため、それによって見張りのパトロールも実現できます。タイムラインを利用すれば、ゲームをより手広くコントロールすることも可能です。たとえば透明の制御用オブジェクトを作成し、特定のタイミングで敵を生成するタイムラインを割り当て、ルームに配置するといったものです。このようにすれば、SF シューティングゲームなどで一定時間おきに敵を発生させることも簡単にできます。

タイムラインを作成するには、アセットブラウザを右クリックして「タイムラインを作成」を選びましょう。ページトップの画像にあるウィンドウが開きます。各領域は次のようになっています。

 

名前名前

タイムラインリソースに名前をつけることができます。英数字または "_" 記号のみ利用でき、コードから参照可能にするため数字から始まる名前をつけることはできません。また、この名前は固有のタイムライン ID となり、DnD™ アクションやコード関数からタイムラインを識別するために利用されます。


モーメントモーメント

ここではモーメント値を設定できます。モーメント値は、タイムライン開始後のゲームステップ数として計算される値です。モーメントの位置が 0 と 60 のタイムラインであれば、モーメント 0 はタイムラインの開始時に作動し、モーメント 60 はそれより 60 ステップ後に発生します。さらにルーム速度が 60 であれば、モーメント 60 はタイムラインが始まってから 1 秒後に発生することになります。モーメント値はダブルクリックするか、右クリックメニューから「変更」（以下のメニューセクションを参照）を選ぶと別の値に変更できます。新しいモーメントを追加するには、エディタの下側にある追加ボタンをクリックしましょう。モーメントが追加されるとコードエディタまたはアクションエディタが開き、モーメントから呼び出す関数や DnD™ アクションを追加できます。


モーメント名モーメント名

タイムラインの内容が分かりやすくなるよう、それぞれのモーメントは名前をつけて区別することができます。この名前はゲームコードやその他の場所とは関係しないただの文字列で、タイムラインを整理したり、動作を把握できるようにするための名前です。モーメントに名前 / 説明を追加するには、次のようにコードブロックの一行目を /// @description 形式にする必要があります。

/// @description Hello World

このように表示されます。

 

タイムラインエディタでは、を押しながらクリックすることで、最初にクリックした位置とその次にクリックした位置の間にあるモーメントをまとめて選択できます。また、 / を押しながらクリックすれば個別選択も可能です。タイムラインのモーメントを右クリックすると、メニューから次の属性を編集できます。

  • 追加: タイムラインに新しいモーメントを追加します。末尾のモーメントの直後に追加されるため、末尾のモーメントが 120 であれば、その次の位置となる 121 に追加されます。モーメントの位置はエディタに表示されているモーメント値をダブルクリックするか、右クリックして「変更」を選択すると変えることができます。
  • 編集: 現在のモーメントを開いて編集します。
  • モーメントの変更: 選択されているモーメントの値を変更します。エディタのウィンドウに表示されているタイムライン値をゆっくりダブルクリックしても同じように変更できます。
  • 説明の変更: 選択されているモーメントの記述名を変更します。エディタのウィンドウに表示されているタイムラインの説明をゆっくりダブルクリックしても同じように変更できます。
  • 複製: 選択されているモーメントを複製し、タイムラインの末尾に続けて追加します。
  • 結合: 2 つ以上のモーメントが選択されている場合、それらを結合するか選択できます。実行すると選択されているモーメントの先頭位置に、すべてのコードとアクションを束ねた単一のモーメントが作成されます。
  • 削除: 選択されているモーメントをすべて削除します。
  • 切り取り / コピー / 貼り付け: タイムラインのモーメントを切り取り・コピー・貼り付けします。 / + "X" 、 / + "C" 、 / + "V" という標準のキーボードショートカットでも同じことが可能です。
  • すべて選択: タイムラインのモーメントをすべて選択します。選択したモーメントはまとめて削除、結合、複製、切り取り、コピーできます。
  • DnD™ / GML に変換: タイムラインのモーメントはプロジェクト作成時に選択する、コードまたは Drag and Drop™ アクションというメソッドのいずれかで作成されますが、この項目を選ぶとプロジェクトの形式を問わず、別のメソッドに変換することができます。
