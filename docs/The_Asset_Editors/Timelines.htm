タイムラインエディタ

多くのゲームでは、一定のタイミングで処理を実行させたい場面があるはずです。Alarm イベントで実現できなくもありませんが、Alarm には最大で 12 個までという制限もあるため、処理が複雑になると上手くいきません。そこで用意されたものがタイムラインアセットです。タイムラインから設定したアクションは（ゲーム内時間の）決められたタイミングで実行され、オブジェクトの各イベント、コードに用意されたすべてのアクションを利用できます。タイムラインはオブジェクトに割り当てることができ、インスタンスがルームに配置または生成されると、指定したタイミングでアクションが順番に実行されます。

たとえば見張りのオブジェクトを作るとして、20 ステップかけて左に移動、10 ステップかけて上に、20 ステップかけて右に、10 ステップかけて下に移動させ、最後に停止させたいとしましょう。これを実装するには、タイムラインを作成してからまず左に動くよう設定します。次にモーメントを 20 で作成し（モーメントとは「瞬間」を意味しており、ゲームステップ、ゲームフレームを言い換えた用語です）、上に動くよう設定します。さらにモーメントを 30 で作成して右に、50 で作成して下に動くよう設定し、最後に 60 で停止させましょう。完成したら、見張りのオブジェクトにタイムラインを割り当てます。それからインスタンスをルームに置けば、想定通りに動くことでしょう。タイムラインでは Alarm も設定できるため、一定時間過ぎてから処理を繰り返し、見張りを巡回させるといったこともできます。また、タイムラインを利用すればゲームをより広い形でコントロールできます。たとえば透明の制御用オブジェクトを作成し、特定のタイミングで敵を生成するタイムラインを割り当ててからルームに配置すれば、SF シューティングゲームなどで一定時間おきに敵を発生させるといったことも簡単に実現できるでしょう。

タイムラインを作成するには、アセットブラウザを右クリックして「タイムラインを作成」を選んでください。ページトップの画像にあるウィンドウが開きます。以下はそれぞれの箇所についての説明です。

 

名前名前

タイムラインリソースに名前をつけることができます。英数字または "_" 記号のみ利用でき、コードから参照できなくなるため数字から始まる名前をつけることはできません。また、この名前は固有のタイムライン ID として扱われ、DnD™ アクションやコード関数からタイムラインを識別するための ID となります。


モーメントモーメント

ここではモーメントの値を設定でき、この値はタイムライン開始後のゲームステップ数として数えられます。モーメントの位置が 0 と 60 のタイムラインであれば、モーメント 0 はタイムラインの開始時に作動し、モーメント 60 はそれより 60 ステップ後に作動します。また、ルーム速度が 60 である場合、モーメント 60 はタイムラインが開始してから 1 秒後に起きることになります。モーメントの値はダブルクリックをするか、右クリックメニューから「変更」を選んで別の値に変更できます（以下のメニュー説明を参照）。新しいモーメントを追加するには、エディタの下側にある追加ボタンをクリックしてください。モーメントを追加するとコードエディタまたはアクションエディタが開き、呼び出したい関数や DnD™ アクションを設定できます。


モーメント名モーメント名

タイムラインの内容が分かりやすくなるよう、モーメントは名前をつけて区別することができます。モーメントの名前はゲームコードやその他の領域とは関わりのないただの文字列で、タイムラインを整理して分かりやすくするためだけの名前です。モーメントに名前 / 説明を追加するには、次のようにコードブロックの一行目を /// @description 形式にしなくてはなりません。

/// @description Hello World

こうすると次のように表示されます。

 

タイムラインエディタでは、 + クリックで最初にクリックした位置とその次にクリックした位置の間にあるモーメントをまとめて選択できます。また、 / を押しながらクリックすれば個別に選択できます。モーメントを右クリックすれば、次のメニューから編集を行うことも可能です。

  • 追加: タイムラインに新しいモーメントを追加します。最後のモーメントの後ろに追加されるため、最後のモーメントが 120 であれば、その次の位置となる 121 に追加されます。モーメントの位置はエディタに表示されているモーメント値をダブルクリックするか、右クリックして「変更」を選ぶと変更できます。
  • 編集: 選択されているモーメントを開いて編集します。
  • モーメントの変更: 選択されているモーメントの値を変更します。エディタに表示されているタイムライン値をゆっくりダブルクリックした場合と同じです。
  • 説明の変更: 選択されているモーメントの記述名を変更します。エディタに表示されているタイムライン説明をゆっくりダブルクリックした場合と同じです。
  • 複製: 選択中のすべてのモーメントを複製し、タイムラインの最後に続けて追加します。
  • 結合: 2 つ以上のモーメントを選択している場合、1 つに結合するかを選択できます。続行すると、選択した中から最初のモーメントの位置にすべてのコードとアクションをまとめた 1 つのモーメントが作成されます。
  • 削除: 選択されているモーメントをすべて削除します。
  • 切り取り / コピー / 貼り付け: タイムラインのモーメントを切り取り、コピー、貼り付けします。 / + "X" 、 / + "C" 、 / + "V" という標準キーボードショートカットでも同じように実行できます。
  • すべて選択: タイムラインのモーメントをすべて選択します。選択したモーメントはまとめて削除、結合、複製、切り取り、コピー可能です。
  • DnD™ / GML に変換: モーメントの形式はプロジェクトの作成時に選択できる GML コード、Drag and Drop™ アクションのいずれかとなりますが、この項目を選ぶとプロジェクトの種類を問わず、モーメントをもう片方の形式に変換できます。
