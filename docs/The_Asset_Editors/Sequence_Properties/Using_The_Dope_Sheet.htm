ドープシートの使い方

ドープシートとは、シーケンスに追加されたトラックのデータが時系列表示される領域です。各トラックの全体像が見れるだけでなく、トラックの配置やパラメーターの変更タイミングなどを編集することができます。

シーケンスに新しいアセットを追加するとトラックパネルにアセットトラックが作成され、ドープシートの再生ヘッド位置にアセットキーが追加されます。このキーはアセットのフレーム長を表しています。

アセットキーを編集するには、キーフレームの先頭か末尾にマウスを移動させ、カーソルが変化してからドラッグしてください。この操作を行うことでキーの開始位置・終了位置を動かせますが、キーの長さも変化します。また、ほかの部分をクリックして横にドラッグするとアセットキーの配置を移動できます。

ドープシートにはアセットキーだけでなく、パラメーターキーも表示されます。パラメーターキーとは、アセットのパラメーター変更シグナルを送る「ポイント」であり、それらの値からシーケンスはアセットに変更を加えます。どのような方式で変更されるかは、トラックパネルのキーにリニア補間を設定したかによって変わります。トラックキーと同じように、パラメーターキーもドラッグしてタイムライン上の配置を変更できます。また、キーを押しながらドラッグすると（カーソルが以下の画像のように変化します）キーを複数フレームに伸ばすことができ、次のキーとの補間が行われるまでキーの値を保たせることができます。

ドープシートでは一般的なキーボードショートカットを利用することもできます。 / を押しながら左クリックでトラックの任意選択、を押しながら左クリックで連続選択ができ、 / +"X" でトラックの切り取り、 / + "C" でコピー、 / + "V" で貼り付けが可能です。

右クリックメニューを開くと、下のほうで説明されている選択項目が表示されます。なお、コマンドを使用してドープシートにキーを貼り付けした場合、キーは再生ヘッドの現在位置に貼り付けされます。

ここまでがドープシートの基本的な使い方となります。それぞれの個所にある役割、そして扱い方を以下でもう少し詳しく説明しましょう。

 

領域の開始 / 領域の終了領域の開始 / 領域の終了

この 2 つのマーカーはクリップ領域の始点・終点です。ゲーム実行時、これらのマーカーによってルーム内に表示されるシーケンスの区間が設定されるほか、シーケンスをエディタでプレビュー再生したときのループ地点となります。これらのポイントは、シーケンスエディタのシーケンス制御から設定できるシーケンス長より大きく、また小さくすることはできません。クリップ領域のマーカーは、目的のフレームまでドラッグすることで位置を変更できます。

 

ズーム操作ズーム操作

ドープシートのズーム状態をチェックできるバーで、倍率の変更もできます。ズーム倍率を変更するには、ウィジェットの端をこのようにドラッグしてください。

なお、ドープシートのズームイン・ズームアウトは / キーを押しながらマウスホイールでも可能です。


アセットトラックアセットトラック

ドープシート内ではアセットトラックが「キー」として表示され、シーケンスのアセットの配置、時間の長さをカラーバーで表します。前述したように、キーは必要に応じて移動させたり、その長さを変えることが可能です。なお、キーをアセットそのものの時間より長くする、たとえば 5 フレーム持続するサウンドのキーを 15 フレームに伸ばした場合、キーの終わりまで再生が繰り返されます。アセットトラックおよびキーは、キーボードショートカットを使用してコピーと貼り付けができます。また、右クリックメニューを開くとさらなる変更が可能です（ページの下部で説明）。

アセットトラックのキーを削除した場合、トラックそのものや、トラックに追加されたパラメーターは削除されず、ドープシートからはアセットキーだけが削除されます。トラックにキーがない場合は、アセットブラウザからアセットを、ドープシートのキーにドラッグすることで新しく追加できますが、元のアセットとまったく別のアセットを追加することはできません（スプライトからキーを削除した場合、そのキーの位置にサウンドアセットを追加することは不可能です）

 

パラメータートラックパラメータートラック

ドープシート内では、パラメータートラックはメインアセットトラックの下にカラーバーとして表示されます。このカラーバーは、パラメーターキーが適用されているトラックを視覚的に示すだけのもので編集できませんが、パラメータートラックの各ポイントは編集可能です（以下のパラメーターキーを参照してください）。

 

再生ヘッド再生ヘッド

タイムラインのプレビュー再生位置です。シーケンスキャンバスにアセットをドラッグしたとき、アセットが追加される位置はこの再生ヘッドの位置から決められ、ほかにもパラメーターの追加、キーの分割など、さまざまな操作における位置の基準となります。キャンバスにある「再生」ボタンをクリックすることで再生ヘッドは前に進みます。ドラッグして特定のフレームに移動させることもでき、そのまま左右に動かせば全体を再生しなくてもざっとプレビューができます。


パラメーターキーパラメーターキー

パラメータートラックのパラメーター変更位置に表示される「ポイント」です。このポイントは移動させたり、冒頭で説明したように 1 フレームより長く引き伸ばすことができ、トラックパネルから直接編集するか、キャンバスのアセットを編集することで値を変更できます（スケール / 位置 / 回転パラメーターのみ）。パラメータトラックのキーは、アセットキーのトラックにもポイント（点）として表示されますが、こちらは編集ができないため、パラメータートラックが折りたたまれている間の目印として扱います。


パラメーターカーブパラメーターカーブ

ドープシートではキーフレームだけでなく、パラメーターカーブを使用してパラメータートラックのパラメーター値を変更できます。このカーブはシーケンスアセットの一部として埋め込みするか、外部アニメーションカーブとしてパラメータートラックに適用できます。詳しい情報はアニメーションカーブの使い方のセクションを確認してください。


 

 

右クリックメニュー

ドープシートのアセットトラックを右クリックすると、次のような選択項目が表示されます。なお、一部の項目はクリックした箇所にキーがなければ表示されません。

  • 切り取り、コピー、貼り付け（キー）: アセットの切り取り / コピー / 貼り付けをします。右クリックメニューから貼り付けをした場合、ドープシートの再生ヘッド位置にキーが貼り付けされます。
  • キーをこの位置に追加: アセットキーを新しく追加します。再生ヘッドの位置ではなく、クリックした位置に近いフレームへ追加されます。
  • キーをこの位置で分割: 選択すると、マウスカーソルの位置でアセットキーを 2 つに分割します。分割されたキーは再生ヘッドの位置ではなく、クリックした位置に近いフレームに作成されます。
  • キーをこの位置に貼り付け: 選択すると、コピー / 切り取りされたキーを貼り付けします。再生ヘッドの位置ではなく、右クリックをした位置に近いフレームに貼り付けされます。
  • キーを削除: 選択中のキーをすべて削除します。
  • 最初にコピー: 選択されているキーをシーケンスの開始位置にコピーします。注意: クリップ領域の開始位置ではありません
  • 最後にコピー: 選択されているキーをシーケンスの終了位置にコピーします。注意: クリップ領域の終了位置ではありません
  • アセットキーを引き伸ばす: 選択中のアセットキーをシーケンスの終了位置（クリップ領域ではありません）、または同じトラック内にある別のキーの位置まで引き伸ばします。
  • アセットキーを削除: 選択中のアセットキーを削除し、同じトラックに別のアセットを追加できるようにします。アセットブラウザからアセットをドラッグすることで追加でき、元のパラメーターを残したまま新しいアセットキーとして編集できます。

ドープシートのパラメータートラックを右クリックした場合は、次のような選択項目となります（クリックした箇所にキーがない場合、あるいはキーフレームとパラメーターカーブのどちらを使用しているかによって項目の表示が切り替わります）。

  • 切り取り、コピー、貼り付け（キー）: パラメーターキーの切り取り / コピー / 貼り付けをします。右クリックメニューから貼り付けをした場合、ドープシートの再生ヘッド位置にキーが貼り付けされます。
  • キーをこの位置に追加: パラメーターキーを新しくパラメータートラックに追加します。再生ヘッドの位置ではなく、クリックした位置に近いフレームへ追加されます。
  • キーをこの位置で分割: 選択すると、マウスカーソルの位置でパラメーターキーを 2 つに分割します。分割されたキーは再生ヘッドの位置ではなく、クリックした位置に近いフレームに作成されます。
  • キーをこの位置に貼り付け: コピー / 切り取りしたキーフレームを貼り付けします。再生ヘッドの位置ではなく、右クリックされた位置に貼り付けされます。
  • キーを削除: 選択中のキーフレームをすべて削除します。
  • 埋め込みアニメカーブに変換: 選択中のパラメータートラックのキーフレームを変換し、埋め込みアニメーションカーブにします（詳しくはアニメーションカーブの使い方のセクションを参照）。アニメーションカーブアセットを使用しているパラメーターカーブを選択した場合、カーブアセットを埋め込みパラメーターカーブとして複製します。
  • 外部アニメカーブを設定: パラメータートラックにアニメーションカーブアセットを割り当てます。詳しい使い方は、アニメーションカーブの使い方のセクションで確認してください。
  • キーに変換: アニメーションカーブを個別のキーフレームに変換します。キャットムル・ロム補間を使用しているカーブを変換すると補間が失われ、カーブのポイント間をリニア補間するキーフレームとなるためご注意ください。また、アニメーションカーブから生成されたキーは必ずフレームの上に位置します。そのため、フレームの中間にポイントがあるカーブをキーに変換すると、もっとも近い整数フレーム上に配置され、カーブの挙動が変わってしまうことがあります。
  • 埋め込みアニメカーブをエクスポート: 埋め込みアニメーションカーブをアニメーションカーブアセットに変換します。変換したカーブはアセットブラウザに表示され、他の場所でも使えるようになります。
  • エディタで開く: ドープシート内でカーブエディタを開き、パラメータートラックに適用されている埋め込みカーブ、または外部アニメーションカーブを編集します。ドープシートのカーブモードに切り替えボタンと同じ機能です。
  • 最初にコピー: 選択されているキーフレームをシーケンスの開始位置にコピーします。注意: クリップ領域の開始位置ではありません
  • 最後にコピー: 選択されているキーフレームをシーケンスの終了位置にコピーします。注意: クリップ領域の終了位置ではありません
  • パラメーターキーを引き伸ばす: 選択されているパラメーターキーを、シーケンスの終了位置（クリップ領域ではありません）、または同じトラック内にある別のキーフレームの位置まで引き伸ばします。パラメーターキーは引き伸ばされたフレーム数、同じ値を保ちます。カーブを選択した場合は、シーケンスの終了位置、または同じトラック内にある別のパラメーターカーブの位置まで引き伸ばします。
  • キーを削除: 実行すると、選択中のパラメーターキーをすべて削除します。
