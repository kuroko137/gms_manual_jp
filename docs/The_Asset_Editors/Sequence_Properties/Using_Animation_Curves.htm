アニメーションカーブの使い方

ドープシートの大きな機能の 1 つは、シーケンスのパラメータートラックにアニメーションカーブを割り当て、パラメーターキーフレームの代わりにパラメーターカーブを作成する機能です。パラメータートラックでは 2 つのタイプのカーブを使用できます。

  • 埋め込みカーブ: シーケンスに埋め込まれ、1 つのパラメータートラック専用となるカーブです。
  • アニメーションカーブ: アニメーションカーブエディタで作成できる、シーケンスから独立したカーブです。

まずはアニメーションカーブアセットの使い方を説明します。簡単に説明すると、カーブを新規作成し、アニメーションカーブエディタでチャンネルを設定してから、ドープシート内のパラメータートラックにドラッグして適用するという手順になります。


また、右クリックメニューから外部アニメカーブに設定を選んでも、選択中のパラメータートラックにアニメーションカーブアセットを追加できます。アセットエクスプローラーが開きますので、使いたいカーブを候補から選びましょう。

アニメーションカーブアセットには、適用させるトラックと同じ数のチャンネルを用意しなくてはなりません。これはパラメータトラックのタイプごとに必要なチャンネルの数が異なるためです。以下はカーブアセットで必要とされるパラメータートラック / チャンネルの一覧です（チャンネルはタイプ別に決められた順番で追加されます）。

  • 位置: チャンネル数 2 （ X、Y ）
  • 回転: チャンネル数 1
  • カラー乗算: チャンネル数 4 （アルファ、赤、緑、青） - 値は 0 から 255 までの範囲に収める必要があります。
  • イメージのインデックス: チャンネル数 1
  • イメージの速度: チャンネル数 1
  • 原点: チャンネル数 2 （ X、Y ）
  • スケール: チャンネル数 2 （ X スケール、Y スケール） - この値は倍率であり、1 を基準値としています。1 より大きくすると拡大し、小さくすると縮小されます。マイナスの値にした場合、イメージを上下 / 左右反転してから拡大縮小します。
  • ピッチ: チャンネル数 1 - 値は 0 から始まり、1 でサウンドのデフォルトピッチになります。
  • 音量: チャンネル数 1 - 値は 0 から始まり、1 で最大音量になります。

デフォルトでは、アニメーションカーブはアセットキーの全長にわたって適用されますが、キーフレームと同じようにタイムラインのカーブをドラッグして配置を変えたり、を押しながらドラッグしてカーブの長さを変更することが可能です（埋め込みカーブも同様）

トラックにパラメーターカーブを追加するもう一つの方法、それが「埋め込み」です。パラメータートラックを右クリックし、「埋め込みアニメカーブに変換」を選ぶことでカーブを追加できます。また、カーブモードに切り替えボタンをクリックすると、トラックがドープシート全体にアニメーションカーブとして表示され、カーブに変換ボタンをクリックしてカーブを追加できるようになります。

キーフレームトラックをカーブに変換すると、ドープシートで編集可能となります。編集のやり方はアニメーションカーブアセットと同一で、ポイントの追加と削除、配置の変更、カーブのタイプ変更（リニア、スムース、ベジェ）、チャンネルの名前変更、色の再割り当てができます。

重要: パラメータートラックに適用したアニメーションカーブアセットもこのように編集できますが、コピーではなくカーブアセットそのものを編集することになるため、同じアニメーションカーブアセットを使用している全トラックに変更が適用されます。アニメーションカーブを独自に編集したいなら、トラックを右クリックしてメニューを開き、埋め込みアニメカーブに変換を選んでください。こうすることでアニメーションカーブアセットが複製され、パラメータートラックにそのトラック専用の独立カーブとして埋め込まれます。あるいはその逆に、トラックを右クリックしてメニューから埋め込みアニメカーブをエクスポートすることで、埋め込みカーブをアニメーションカーブアセットに変換することも可能です。この場合は埋め込みカーブが新しいアセットに変換され、普通のカーブと同じように編集したり、他のパラメータートラックへ適用できるようになります。

この画像は埋め込みアニメーションカーブエディタの表示例で、アニメーションカーブエディタと非常によく似ています。以下のセクションは各部分についての詳細です。

 

カーブ名カーブ名

外部アニメーションカーブアセット、または埋め込みカーブのパラメータートラック名です。名前は変更可能ですが、外部アニメーションカーブアセットに行った場合はパラメータートラックではなく、アセットそのものの名前が変わります。

 

カーブタイプカーブタイプ

カーブタイプアイコンをクリックすると、チャンネルで使用するカーブのタイプを以下から選択できます。

  • リニア: チャンネルのポイント間をリニア補間します。
  • スムース: キャットムル・ロム補間によって、チャンネルのポイント間をなだらかに進行させます。
  • ベジェ: ベジェ補間によって、チャンネルのポイント間をなだらかに進行させます。ポイントの「ハンドル」を操作することでカーブの形状を調整可能です。

 

チャンネルリストチャンネルリスト

チャンネルリストでは、パラメーターカーブの構成チャンネルを編集できます。それぞれのチャンネルには別々の名前が付けられ（英数字と "_" 記号のみ利用可）、左側の色見本をダブルクリックするとチャンネルの色も設定できます。名前については、チャンネルをゆっくりとダブルクリックすれば変更可能です。また、右クリックで開けるメニューからも名前変更、色の変更、チャンネルのコピーができます。名前の横にある矢印アイコンをクリックした場合は、チャンネルが展開されてカーブの各ポイントが表示されます。これらのポイントは、現在値をクリックして新しい値を入力することで直接編集ができます。ドープシートの「目」アイコンは、クリックするとチャンネルの表示を切り替えます。

 

変換操作変換操作

ページの前半で説明したキーフレームをパラメーターカーブに変換するだけでなく、パラメーターカーブをキーフレームに変換するボタンもあります。変換を行う場合、2 つの注意点を留めておいてください。まず、キーフレームはフレームの「中間」に配置できないため、カーブの値を隣接フレームに丸めなくてはならず、結果としてパラメータートラックの挙動が変化してしまう可能性があります。次に、パラメーターカーブにキャットムル・ロムまたはベジェ補間が使用されている場合、補間が打ち消され、キーフレームはリニア補間によって補間されます。

 

カーブメニューカーブメニュー

このメニューには、エディタの別の箇所にある選択項目も含まれています。

  • 範囲の設定: パラメーターカーブの縦の範囲を設定します（以下のチャンネル範囲を参照）。
  • カーブに変換: 選択したパラメータートラックのキーフレームを、パラメーターカーブに変換します。
  • キーに変換: 選択したパラメータートラックのパラメーターカーブを、キーフレームに変換します。
  • カーブを埋め込み: パラメータートラックに使用されているアニメーションカーブアセットをトラックに埋め込みします。
  • 外部アニメカーブを設定: 埋め込みパラメーターカーブをアニメーションカーブアセットに置き換えます。

 

チャンネル範囲チャンネル範囲

チャンネル範囲ボタンをクリックすると、チャンネルエディタに表示される縦の範囲を設定できます。デフォルトでは-1 から 1 となっており、次のウィンドウで設定を行います。

数値が変更されると、チャンネルエディタのデータ表示が変化します。これらの数値は-1 から 1 より低い / 高い数値にできます。

 

チャンネルエディタチャンネルエディタ

チャンネルエディタにはパラメーターカーブの各チャンネルが視覚的に表示されます。デフォルトではすべてのチャンネルが表示されますが、チャンネルリストで選択中の、エディタに強調表示されているチャンネルのみ編集できます。チャンネルカーブ上のポイントはドラッグして配置を変更できますが、始点・終点のポイントは縦方向にしか移動できず、横方向の値が 0 または 1 となるためご注意ください。

メインチャンネルの線にマウスカーソルを合わせ、カーソルが「ポイント追加」ツールに変化しているときにクリックすると、チャンネルカーブに直接ポイントを追加できます。追加したポイントを編集するにはマウスでドラッグするか、チャンネルリストでチャンネルを展開し、値を変更してください。

エディタ上をクリックしてドラッグすると範囲選択、 / を押しながらポイントをクリックすると任意選択ができます。選択したポイントをドラッグすると、複数のポイントをまとめて動かすことができます。

パラメーターカーブの編集が終わったら、カーブモードに切り替えボタンをクリックすることでドープシートを通常表示に戻せます。
