トラックパネル

シーケンスの動作はトラックパネルに追加されたアセットとパラメーターによって制御されます。以下のいずれかの手順でエディタにトラックを追加することができます。

  • アセットをシーケンスキャンバスにドラッグ
  • アセットをトラックパネルにドラッグ
  • トラックパネルの下にある「アセットを追加」ボタンをクリック

3 番目の手順ではアセットエクスプローラーが開きますので、そこから追加したいアセットを選択しましょう。

どの手順で追加した場合も、アセットには新しいアセットトラックが与えられます。このトラックはシーケンスに追加されたアセットのメイントラックであり、ドープシートの再生ヘッド位置に作成されるアセットキートラックと対応しています。

ドープシートのアセットキーの色は、トラックパネルの端にあるカラーバーをクリックすると変更できます。クリックすると開くカラーピッカーから、ドープシートのキートラックに割り当てる色を選択しましょう。

トラックエディタのアセットにはパラメータートラックを追加できます。このトラックはさまざまな属性（パラメーター）を操作するためのもので、時間によって変化させることができます。利用可能なパラメーターについてはこのページの下部にある「パラメータートラックの追加」セクションで詳しく説明されていますが、どういうパラメーターがあるのかここで簡単に紹介しておきましょう。

  • すべてのアセット: 位置, 回転
  • サウンド: ピッチ, ボリューム
  • シーケンス: 原点, スケール
  • スプライトとオブジェクト: 原点, スケール, 画像のインデックス, 画像の速度, カラーブレンド

エディタにパラメータートラックを追加するには、トラックパネルから「変更の自動記録」を有効にします。有効にしたままキャンバスのアセットに変更を加えると、再生ヘッドの位置にパラメーターキーが追加されます。以下の画像がその例です。

メニューの項目を選ぶ、または IDE から直接記録を行うと、メインアセットトラックの下にパラメータートラックが 1 つ以上作成され、パラメーターキーがドープシートに追加されます。このキーの色はメインアセットトラックと同一になります。

メインアセットトラックにパラメータートラックを追加すると、アセットトラックのフレーム長をあらわすカラーバーがドープシートに追加されます。また、パラメーターを変化させるポイントにはパラメーターキー（キーフレーム）が追加されます。上記の画像において、フレーム上にある「点」のマークがパラメーターキーです。トラックにパラメーターキーが与えられている場合、デフォルトでは参照されているパラメーターの値が他のポイントとの間で補間（リニア補間）されます。たとえば回転のパラメーターが 0 フレーム目で 90° 、30 フレーム目で 180° に設定されている場合、15 フレーム目では 135° になります。パラメータートラックを選択したまま再生ヘッドを左右に動かすと、値がどう変化するのかよく分かるでしょう。

パラメータートラックはデフォルトで他のキーとリニア補間されますが、以下のように右クリックメニューから補間を無効にすることができます。無効にするとパラメーターの値は時間をかけて推移しなくなり、指定したフレームに達した瞬間に変化します。

右クリックメニューにはパラメータートラックの推移方式を変更する以外にも、以下のような選択項目があります。

  • 切り取り, コピー, 貼り付け: アセットやパラメータートラックを切り取り / コピー / 貼り付けします。
  • トラックの複製: 選択中のトラックを複製します。アセットトラックが選択されている場合、関連するパラメータートラックも複製されます。
  • トラックを削除: 選択中のトラックを削除します。アセットトラックを削除した場合、関連するパラメータートラックも削除されます。なお、パラメータートラックは個別に削除することができます。
  • グループ: トラックパネルの余白で実行すると、グループフォルダが作成されます。このフォルダはアセットを整理するためのものです。グループに入れたいアセットはこのフォルダにドラッグしてください。トラックを選択したまま選択項目を実行すると、グループフォルダを作成してから選択中のアセットをそのフォルダに入れます。
  • 名前変更: アセットトラックまたはグループフォルダの名前を変更します。

アセットとパラメータートラックには、トラックとグループの親になれるという重要な機能があります。トラックパネルのアセットトラックを別のトラックにドラッグすると、そのアセットは子となり、ドラッグ先のアセットは親となります。親アセットトラックに割り当てられているパラメータートラックは子アセットトラックに自動適用されるため、たとえば以下のように複数のスプライトを 1 つの位置アセットトラックで移動させることも可能です。トラックパネルの下部にあるボタンをクリックするとグループフォルダを作成できます。グループ化するアセットは作成したグループフォルダにドラッグしましょう。グループ化したアセットは元のパラメータートラックを保持していますが、グループそのものにもパラメータートラックを追加でき、グループ内のアセット、パラメータートラックはそれらのパラメータートラックによる影響を受けます。

トラックパネルではキーボードショートカットを利用できます。 / を押しながら左クリックで複数トラックの任意選択、を押しながら左クリックで連続選択ができ、 / + "X" で切り取り、 / + "C" でコピー、 / + "V" で貼り付けができます。

ここまでがトラックエディタの基本的な使い方となります。以下のセクションでそれぞれの個所の役割、使い方についてより詳しく説明していきます。

 

向きを変更向きを変更

向きを変更ボタンを使うと、シーケンスエディタの各部分の表示方法を変更できます。デフォルトではキャンバスビューが上部に、トラックエディタとドープシートが下部に表示されていますが、このボタンをクリックすると異なるレイアウトに順番で切り替わります。また、設定ボタンをクリックするとメニューから好きなレイアウトを選ぶこともできます。使用できるレイアウトは次の通りです。

ウィンドウを分割

シーケンスエディタが 2 つのウィンドウに分かれます。IDE のメインウィンドウにはキャンバスビューが全体表示され、新しいウィンドウにはドープシートとトラックエディタが表示されます。ボタンをもう一度クリックするとレイアウトが切り替わり、新しいウィンドウが消えます。

シーケンスエディタのレイアウトは、リセットオプションをクリックすることでいつでも元に戻せます。レイアウトをリセットするとエディタがデフォルトの状態に戻り、閉じられていた部分も開き直されます。

 

トラック検索トラック検索

検索ボックスを使うことでシーケンス内のトラックを検索し、編集したいトラックをすぐに見つけることができます。検索ボックスに文字を入力すると、入力された文字列を含むトラックの件数が表示されるほか、検索ボックスの横にある矢印の色が変化します。この矢印をクリックすると、検索条件と合致するアセットを順番に選択することができます。

 

アセットトラックアセットトラック

シーケンスに追加されたアセットにはトラックが与えられ、トラックパネルのリストに加わります。パラメーター付きのアセットトラックの場合、パラメータートラックが下に連なって表示されます。トラック名の左にある矢印アイコンをクリックすると、パラメータートラックを展開・収納できます。また、トラックをゆっくりダブルクリックすると好きなトラック名に変更できます。トラック名には命名制限がないため、半角スペースやアルファベット以外の文字列を使ったり、あるいは数字から始まる名前にすることも可能です。

使用可能なパラメータートラックの種類については、このページの「パラメータートラックを追加」セクションにおいて詳しく説明されています。

 

キー操作部キー操作部

パラメータートラックが追加されていると、特定のフレームにパラメーターキーを追加・削除してパラメーターの値を設定できます。「キーを記録」または「キーを削除」ボタン（「トラックのキー操作部」セクションに詳細あり）をクリックするか、パラメーターのキー操作部によって設定できます。これらの操作を行なうには、ドープシートの再生ヘッド位置を編集したいフレームに合わせ、パラメータートラックの左にある丸い「キー」アイコンをクリックしてください。フレームにキーが割り当てられていないとアイコンの丸い部分が「空白」となり、クリックすると新しいキーが追加されてパラメーターの値を編集できます。フレームにキーが割り当てられているとアイコンの丸い部分が「色つき」になり、クリックするとフレームからキーが削除されます。

上記の画像を見てお気付きかもしれませんが、パラメーターキーの横にある矢印ボタンをクリックすると、再生ヘッドを動かさなくてもドープシートのキーフレーム間をすばやく移動することができます。

 

パラメーター値パラメーター値

パラメータートラックが追加されている場合は、ドープシートの再生ヘッドを変更したいキーのあるフレームに置き、各キャンバスウィジェットから編集することでアセットの位置、回転、スケール、原点といった値を変更できます。変更はそのフレームのキーにのみ反映されます。すべてのパラメーターをこの方法で編集できるわけではなく、より正確に調整しなくてはならない場合もあるはずですので、そういったときはトラックを直接クリックしてパラメーターキーを編集し、タイプごとに必要な値を設定しましょう。

一部のパラメータートラックは、値の横に「リンク」アイコンが表示されます。このアイコンをクリックすると値のリンクが有効 / 無効となり、1 つの値を編集するだけですべての値を同時に変更できます。

 

イベントイベント

通常のオブジェクトと同じくシーケンスにはイベントを追加でき、実行させるコードを割り当てることができます。コードは引数を受け付けないスクリプト関数として割り当てられ、イベントが発生したタイミングで呼び出されます。

イベントの追加ボタンをクリックすると、次のウィンドウからイベントを追加できます。

以下は追加できるイベントの一覧です。

  • Create - シーケンスのインスタンスが最初に生成されたときに起きるイベントです。ルームエディタによって配置されたシーケンス内で最初に起きるイベントであり、ルームに入ったタイミングで発生します。
  • Destroy - シーケンスのインスタンスが破棄されたときに実行されるイベントです。以下の Clean Up イベントより先に実行されます。
  • Clean Up - シーケンスのインスタンスをルームから除去する全イベントの後に呼び出されるイベントです。つまりシーケンスの破棄、ルームの終了、ゲームの終了によって作動するイベントとなり、インスタンスがゲームから除去されたときに任意の処理を実行させる役割を持っています。シーケンスのインスタンスが layer_sequence_destroy() などで破棄された場合、このイベントは上記の Destroy イベントより後に呼び出されます。
  • Begin Step / Step / End Step - シーケンスが存在している間、ゲーム内での 1 ステップ（フレーム）ごとにチェックされるイベントが Step イベントであり、Begin 、Step 、End という 3 つの段階に分けられています。ほとんどの場合は普通の Step イベントで問題ありませんが、どのコードをどのタイミングで実行させるかをもう少しコントロールしたい場合もあるでしょう。そのために用意されているイベントが Begin と End イベントです。この二つのイベントはゲーム内での 1 ステップ（フレーム）ごとに、必ず決められた順番で作動します。注意！シーケンスが一時停止されている間、Step イベントは作動しません。シーケンスが再開された場合、再生を再び始めた直後のフレームで作動します。また、再生ヘッドの方向によってイベントの順番が変わることはありません。シーケンスが逆再生されている場合でも、Step イベントは Begin 、Step 、End の順に実行されます。
  • Async Event: オブジェクトの Asynchronous System Event に相当するイベントです。注意！シーケンスが一時停止されている間、このイベントは作動しません。

各イベントには、イベントの作動時に呼び出す関数を 1 つ割り当てることができます。関数の割り当てはイベントの入力ボックスから行ない、矢印ボタンをクリックするとスクリプトエディタを開いて関数を編集することもできます。また、一番下にある新規作成ボタンをクリックすると「定型」関数入りのスクリプトリソースが作成され、すべてのイベントに割り当てられます。関数の名前は自由に変更できますが、すでに定義されている名前は使用しないようにしてください。不要となった関数は定義を削除するといいでしょう。なお、シーケンスイベントで使用される関数は引数を受け取らないため注意してください。

シーケンスイベントはコードから追加・編集することも可能です。詳しくはこちらを参照してください。

 

キーフレーム操作部キーフレーム操作部

キーフレーム制御ボタンはドープシートにおける特定のキーを追加・削除・編集できるボタン群です。以下がその一覧となります。

  •  このボタンは「選択キー分割」ボタンです。ドープシートのパラメーターキーを 1 つ以上選択している状態でこのボタンをクリックすると、タイムラインの再生ヘッド位置でキーを 2 つに分割します。
  •  - このボタンは「選択キー削除」ボタンです。ドープシートのキーを 1 つ以上選択した状態でこのボタンをクリックすると、選択中のキーを削除します。
  •  - このボタンは「キーの新規記録」ボタンです。ドープシートのアセットキーを選択した状態でこのボタンをクリックすると、アセットトラックのパラメータートラックに新しいパラメーターキーを作成し、タイムラインの再生ヘッド位置にパラメーターキーのポイントが追加されます。
  •  - このボタンは「変更の自動記録」ボタンです。このボタンが押されているとキャンバス上のアセットに加えた変更が自動記録されるようになり、ドープシートの再生ヘッド位置に適切なパラメータートラックおよびパラメーターキーを追加します。たとえば、再生ヘッドを 0 フレーム目から 10 フレーム目に動かしてからキャンバス上のアセットを 100 ピクセル右に移動させると、パラメーターキーが 0 フレーム目（初期位置）と 10 フレーム目（再生ヘッド位置）に追加されます。この状態でシーケンスの「再生」ボタンを押すと、10 フレームかけてアセットが 100 ピクセル右に移動します。

表示 / ロックの切り替え表示 / ロックの切り替え

アセットトラックとパラメータートラックのどちらにもトラックの表示とロックを切り替えるボタンがあります。トラックの表示を無効にすると、ルームでシーケンスを再生する場合も、シーケンスキャンバスで再生テストをする場合もそのトラックが表示されなくなります。また、アセットトラックの表示を切り替えるとパラメータートラックの表示も切り替わります。トラックのロックは余計な編集を防ぐためだけの機能です。なお以下のように、非表示にしたトラックはドープシートに透過して表示され、ロックしたトラックにはオーバーレイが重なって表示されるため、トラックの状態をぱっと見るだけで確認できます。

 

トラックの色トラックの色

作成されたアセットトラックには、ドープシート上での見分けがつくように IDE からランダムな色が割り当てられます。この色は自由に変更でき、トラックのカラーバーをクリックして開けるカラーピッカーから好きな色を選択できます。選択した色はアセットトラックの関連パラメータートラックにも自動で適用されますが、パラメータートラックの色は個別に変えることもできます。アセットトラックの色を再変更した場合は、パラメータートラックの色も上書きされるためご注意ください。

 

パラメータートラックの追加パラメータートラックの追加

パラメーター追加ボタンをクリックすると、選択中のアセットトラックに 1 つのパラメータートラックを追加できます。このページの冒頭で説明したように、パラメータートラックにはアセットとの関係に応じてさまざまな種類があります。以下は使用可能なすべてのパラメータートラックの一覧です。どのアセットに関係しているのか、どのアセットに適用されるのかも記載されています。

  • 位置: 4 つのシーケンスタイプ（スプライト、オブジェクト、サウンド、シーケンス）のすべてに追加できるパラメータートラックで、シーケンス内アセットの空間位置を設定します。X と Y の値を必要とします。
  • 回転: 位置パラメーターと同じく、回転パラメーターはすべてのシーケンスアセットで使用でき、アセットの回転を角度で指定します。GameMaker Studio 2 では反時計回りで回転が計算されるため、0° が右、90° が上、180° が左、270° が下となることに留意してください。
  • 原点: 原点パラメーターはスプライト、オブジェクト、シーケンスアセットで使用でき、アセットの原点の X と Y 位置を設定します。原点に変更を加えると、アセットの位置、回転、スケールがどう適用されるかに影響します。
  • スケール: スケールパラメーターはスプライト、オブジェクト、シーケンスアセットで使用でき、アセットの水平・垂直スケールを設定します。このパラメーターは IDE および実行時におけるアセットの表示を変化させます。スカラー値で計算され、100%にするとアセットのデフォルトサイズに、50%にすると半分のサイズに、200%にすると倍のサイズになります。水平・垂直のスケール値は「リンク」ボタンをクリックして連動させることができ、片方の値を変更したときにもう片方の値も自動で変更されるようになります。
  • 画像のインデックス, 画像の速度: スプライトとオブジェクトアセットでのみ使用可能なパラメーターで、現在のアニメーションフレーム（詳しくは image_index のページを参照）およびアニメーション速度を変更します。アニメーション速度は FPS から決められ、スプライトエディタでスプライトの値を設定するときと同じように設定できます。
  • カラー乗算: スプライトとオブジェクトアセットで使用可能なパラメーターで、スプライト描画時のブレンドカラーを変更します。image_blend の値を設定することと同一であり、設定した色がスプライトの色に「乗算」（もといブレンド）されます。値が白色（#ffffffff）であれば、ブレンドが行われないことになります。このタイプのトラックを扱うときは、16 進数の値を直接入力する、または GameMaker Studio 2 のカラーピッカーから色見本をクリックして色を設定できます。また、この色は ARGB 形式で指定されるため、こうしたパラメータートラックを利用してアルファ値を設定することができます（カラーピッカーでもアルファ値を設定可能です）。これはスプライトの image_alpha を設定することと同一です。
  • 音量: 音量パラメーターはサウンドアセットトラックでのみ使用可能なパラメーターで、サウンドのゲインを設定します（実際にはサウンドアセットのゲインではなく、シーケンスのアセットトラックが利用するエミッタのゲインを変更することになります）。ボリュームの値はパーセントで計算され、0%で無音、100%で最大音量となります。
  • ピッチ: ピッチパラメーターはサウンドアセットトラックでのみ使用可能なパラメーターで、サウンドのピッチを設定します（音量と同じようにサウンドアセットのピッチではなく、シーケンスのアセットトラックが利用するエミッタのピッチを変更することになります）。ピッチの値はパーセントで計算され、100%にすると「本来」のピッチとなり、100%より少さくするとピッチが低く、100%より大きくするとピッチが高くなります。
  • 減衰: 減衰パラメーターはサウンドアセットトラックでのみ使用可能なパラメーターです。現時点では未実装となります。

 

トラックアクショントラックアクション

トラックエディタの下にはトラックアクションを実行するボタンがあり、次のように分かれています。

  •  - このボタンをクリックすると、アセットエクスプローラーからアセットを選択して新しいトラックとします。選択されたアセットからアセットトラックが作成され、ドープシートの再生ヘッド位置にフレームが追加されます。
  •  - トラックエディタのアセットトラックを 1 つ以上選択してからこのボタンをクリックすると、選択中のアセットトラックを複製します（中にあるパラメータートラックも複製されます）。
  •  - このボタンをクリックすると、クリッピングマスクグループが作成されます。このグループは特殊なグループであり、マスクにするスプライトとマスクで隠すスプライトを指定します。詳しくはこちらを参照してください。
  •  - このボタンをクリックすると、トラックエディタに新しいグループフォルダが作成されます。このフォルダはトラックを整理するためのもので、ドラッグして入れたトラックはグループに追加されます。もう一度外にドラッグすればグループからの削除も可能です。グループフォルダにアセットを追加すると、自動的にフォルダの色が割り当てられるためご注意ください。なお、必要に応じてトラックごとに色を変更できます。
  •  - 見てのとおり、選択中のトラックを削除するボタンです。アセットトラックを選択した場合、そのトラック内のパラメータートラックもすべて削除されますが、パラメータートラックは個別に選択することでアセットトラックごと消さずに削除可能です。空のグループフォルダを選択した場合、そのフォルダを削除します。ただしフォルダに 1 つでもアセットが含まれていると警告が表示され、そのまま削除すると中にあるアセットもすべて削除されます。
