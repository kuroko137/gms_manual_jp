シーケンスキャンバス

プロジェクトのシーケンスを作る上で、大部分の作業はシーケンスエディタのキャンバスで行われます。キャンバスとはシーケンス要素が入れられる場所であり、シーケンスがルームに追加されたときにどう表示されるかを決める領域です。最初にシーケンスを作成すると、キャンバスのデフォルトサイズは 1366px * 768px に設定されます。このサイズは、ツールボックスのキャンバスフレーム切替ボタンの横にあるをクリックすると変更可能です。

キャンバスのサイズはシーケンスの構成アセットを配置するうえでの視覚的範囲となりますが、このフレームはいわば大きさの目安であり、レンダリングの結果には影響しません。たとえば、スプライトアセットがフレームの外に配置されているシーケンスをルームに追加した場合でも、そのスプライトアセットはレンダリングされます。

フレームには参考画像を割り当てることができ、シーケンスのアセット配置を合わせるためのガイドとして扱えます。参考画像はキャンバスのフレームサイズと同じくキャンバスオプションから設定でき、設定した画像はシーケンスキャンバスの背景画像となります。このオプションを利用することで、たとえばプロジェクトのアーティストが制作した UI モデルに合わせて配置を行うということができます。

参考画像はプロジェクトには含まれず、シーケンスに追加してもレンダリングされないためご注意ください。

キャンバスの設定ができたら今度はアセットを追加しましょう。アセットはアセットブラウザからキャンバスにドラッグすると追加できます。アセットを追加すると、トラックパネルにアセットトラックが作成されます。

キャンバスには以下のアセットを追加できます。

  • スプライト
  • オブジェクト
  • サウンド
  • シーケンス

必要なアセットを追加できたら、アセットをそれぞれ初期位置に配置しましょう。さらに変形操作ギズモを使えばさまざまな変形も可能です。変形は 2 つの方法で行うことができます。

  • キャンバスに配置したアセットの端にマウスカーソルを合わせてドラッグ（カーソルの表示が変形操作に合わせて切り替わります）:
  • アセットの原点にある変形ギズモ（ツールボックスから変更可能）を使用:

こうした初期変形は必要があれば設定してください。すべて終えたら、今度はトラックパネルのアセットにパラメーターキーを追加しましょう。簡単な例として、基本的な位置変形を紹介します。アセットはもう開始位置に配置されているはずですので、移動を終わらせるフレームまでドープシートの再生ヘッドを動かし、それからアセットの配置をもう一度変更してください。

上記の画像では、新しい位置がトラックに自動で記録されています。これはトラックパネルから "変更の自動記録" を有効にしたためです。キャンバスビューに表示されている点線はアセットの時間による動きを示しています。自動記録を有効にしていない場合でも、再生ヘッドの位置を決めてからアセットを動かし、"キーを新しく記録" ボタンを押すことでポイントを追加できます。

もう一つお伝えしておきたい点は、キャンバスからスマートガイドを追加できるということです。スマートガイドとはキャンバスに追加できる目安線のことで、アセットはこの線に沿って配置することができます。デフォルトのキャンバスでは、中心点に水平 / 垂直のガイドが表示されます。さらにシーケンスキャンバスの端には、キャンバスのサイズと同じ大きさの長方形ガイドが表示されます。これらはルームに追加した後のシーケンスではレンダリングされません。あくまで配置の参考としたり、位置を揃えるためのものです。また、以下のように定規をクリックしてキャンバスまでドラッグすれば独自にガイドを追加できます。

ガイドは定規のところまでドラッグして戻すと削除できます。シーケンスにガイドが追加されていると、アセットを動かしてバウンディングボックスまたは原点を近づけるとガイドに "スナップ" させることができます。同時にスマートガイドが表示され、他のアセットとの距離関係や、同じガイドまたは同じ位置上にあるアセットを示してくれます（それぞれ別の色で）。

 

 

右クリックメニューとナビゲーション

キャンバスビューは通常のワークスペースと同じように基本的な操作でナビゲート可能です。たとえば中央マウスボタン（または + ）でドラッグするとキャンバスの移動、マウスホイールで垂直スクロール、 / キーとマウスホイールでズームインとズームアウトができます。

アセットの選択は左クリックで行い、 / を押しながらクリックすれば複数のアセットを選択できます。また、複数選択はを押しながらドラッグでも可能です。この操作をするとキャンバスに「ボックス」が表示され、その中に入れたアセットが選択されます。

選択されたアセットは切り取り（ / +"X" ）、コピー（ / + "C" ）、貼り付け（ / + "V" ）ができます。

キャンバスで複数のアセットを選択すると、それぞれのアセットにバウンディングボックスが表示されるほか、そのすべてを合体させた大きなバウンディングボックスが表示されます。画像の左上にあるアイコンからボックスの動作を変更できます。

丸いアイコンをクリックすると、選択アセットの全体バウンディングボックスの原点が変形操作ギズモに切り替わります。箱型のアイコンをクリックすると、それぞれのアセットごとに操作ウィジェットが表示されます。グループ化されたアセットは、単一のアセットと同じようにコピー / 切り取りなどの操作が可能です。

複数のアセットが重なっていて、その中から 1 つのアセットをスムーズに選択したい場合があります。マウスカーソルをアセットの上に置いてみましょう。すると項目ツールボックスが開きます。カーソルの下にある項目がアイコン化されてリスト表示されますので、選択したい項目をクリックしましょう。

シーケンスのアセットを右クリックすると、ほかにも多くの選択項目を選べます。

メニューの選択項目:

  • 記録: アセットのパラメーターキーを記録できます。カーソルを置くと、さらにスケール、回転、位置の記録が表示されます。
  • グループ: トラックパネルに新しいグループフォルダを作成し、選択されているアセットをそのフォルダに入れます。
  • 名前変更: クリックすると、トラックパネルのトラックがハイライト表示されて名前の編集に入ります。
  • シートを探す: クリックすると、選択されているアセットのキーがドープシート上でハイライト表示されます。
  • 他を探す: クリックすると、同じタイプのアセットをリストから個別に選択できます。
  • 選択: クリックすると、クリックした位置で重なっているすべてのアセットがリスト表示されます。リストから編集したいアセットを個別に選択できます。
  • キーを削除: 選択した場合、ドープシートの現在位置からアセットトラックキーが削除されます。トラックそのものはパラメータートラックとともに残るため、アセットブラウザからドープシートにアセットをドラッグすることで、同じトラックに別のアセットを再追加できます。
  • トラックを削除: アセットトラック（および割り当てられているパラメータートラック）をトラックパネルから削除します。ドープシートの関連キーもすべて削除されます。
  • エディタで開く: アセットをそれぞれのアセット用のエディタで開きます。
