フォントエディタ

ゲーム内でテキストを描画しようとすると、標準の Arial 12 ポイントフォントで描画されます。しかし、より目を引くような、個性あるビジュアルのテキストにするには、別のフォントを使用しなくてはならないでしょう。コンピューターにある別のフォントを使用するには、GameMaker Studio 2 のフォントアセットを作成しなくてはなりません。フォントアセットは、アセットブラウザを右クリックして「フォント作成」を選ぶと作成できます。作成したフォントアセットにはコンピューターにある特定のフォントを指定可能で、アクションまたはコードから描画用のフォントに設定することでゲーム内でも使用できます。

フォントアセットが作成されると、フォントエディタのウィンドウが開き、次のような領域が表示されます。

詳細詳細

フォントに固有の名前（英数字および "_" 記号のみ利用可）をつけたり、使用フォントの指定ができます。コンピューターにインストールされているフォントのみ指定でき、また GameMaker Studio 2 ではライセンス上の問題から、デフォルトではプロジェクトファイルにフォントファイルそのものを保存せず、一連のビットマップグリフとしてフォントのテクスチャページをレンダリングします。このテクスチャページはテクスチャグループに割り当てられ、グループオプションから特定のグループを指定できます。

マーケットプレイスのアセットを作成するなど、何らかの理由でフォントファイルそのもの（*.ttf のみ対応）をプロジェクトに含めて配布したい場合は、プロジェクトにコピーをチェックしてください。エクスプローラーが開き、指定したフォントファイルをプロジェクトにバンドルされるファイルとして追加できます。

注意！ 必ず自分のプロジェクトで使用または再配布することが法的に許可され、ライセンス上の問題がないフォントのみを利用するようにしてください。

 

プリセットプリセット

ここではサイズ（ポイントで指定、デフォルトは 12pt ）のほかに、フォントの太字、斜体、アンチエイリアスを適用するかを設定できます。右側のプレビューウィンドウには指定したサイズと書体で、実際のゲームとほぼ同じようにテキストが表示されます。


範囲操作部範囲操作部

フォントリソースの作成時、そのフォントのグリフをすべて含める必要はほとんどの場合ありません。そのため GameMaker Studio 2 では選択されたフォントのうち、数字、小文字、大文字、特定の句読点をカバーできる 32 から 127 文字目が使用されます。ですがゲームを最適化するために使用する文字範囲を変えることも可能で、たとえば数字だけを選択すれば、数字だけを使用することができます。手順としては、まず削除ボタンをクリックして現在の範囲を消去し（プレビューウィンドウをクリックして範囲を選択しなければならない場合もあります）、追加ボタンをクリックして新しい範囲を追加してください。すると次のウィンドウが隣に開きます。

このウィンドウには次のボタンがあり、クリックするとフォントを標準範囲に設定できます。

   - 通常範囲: 32 から 127

- ASCII 範囲: 0 から 255

- 数値範囲: 数値 10 文字のみ

- 文字列: 大文字および小文字の文字列のみ

異なる範囲にするには、文字列範囲のボックスに最初と最後の文字番号を入力してください（範囲内にない文字はスペースに置き換えられます）。上記のボタンと入力ボックス以外にも、次のように便利なボタンが 2 つあります。

  • コードから検索: クリックして OK を押すと、GameMaker Studio 2 側でゲームコードの文字列を自動解析し、ゲームの全テキストをカバーできる文字範囲を新しく作成します。なお、すべての文字列（ "" または''の中身）を調べるため、文字範囲にファイルの名前が混ざってしまう場合があります。
  • ファイルから検索: 上にある「コードから検索」と同じような機能です。クリックすると参照先のファイルを指定できますので、決まったら OK を押してください。GameMaker Studio 2 によってファイルが解析され、含まれているテキストから文字範囲が作成されます。

なお、テキストの範囲はプレビューウィンドウに直接テキストグリフを追加・削除することで手動編集も可能です。

 

テクスチャ操作部テクスチャ操作部

上記（フォント詳細のセクション）で言及したように、デフォルトではすべてのフォントグリフはグラフィックリソースに変換され、テクスチャグループ内のテクスチャページとしてレンダリングされます。ゲームのコンパイル時、一時テクスチャに追加されたフォントから必要なフォントグリフをすべて含めた 1 枚の画像が作成され、メインテクスチャグループに追加した上でエクスポートされます。並んでいるボタンをクリックすると、フォントテクスチャの生成方法を調整したり、新しいフォントテクスチャを再生成することができます。

オプションボタンをクリックした場合は、フォントテクスチャの詳細設定が開きます。

これらのオプションを設定すると、ゲーム内でのフォントの見栄えを根本的に変えられる場合もありますが、すべてのフォントが同様の設定を必要とするわけではありません。以下のオプションは、フォントを最大限カスタマイズするために用意されたものです。

  • ヒンティング: フォントヒンティング（またはインストラクション）とは数学的な命令を通して、ラスタライズされたグリッドに並ぶようアウトラインフォントの表示を調整することです。低い解像度では、テキストをはっきりと読みやすくするためヒンティングを利用することが重要になってきます。ヒンティングのアルゴリズムを変更して、なるべく見栄えを良くすることも可能です。デフォルトでは「通常」のアルゴリズムとなっています。
  • グリフ設定: 以下にある操作を有効 / 無効にすることで、グリフの読み込み方法を変更するオプションです。
    1. ヒンティングを無効: フォントのネイティブヒンティング、自動ヒンティングを無効にします。
    2. 自動ヒンティングを無効: 自動ヒンティングを無効にし、フォントのネイティブヒンティングが利用できる場合はそちらを利用します。このオプションは「自動ヒンティングを優先」を打ち消すためご注意ください。
    3. 自動ヒンティングを優先: フォントのネイティブヒンティングよりも GameMaker Studio 2 の自動ヒンティングを優先させます。このオプションは「自動ヒンティングを無効」、および「ヒンティングを無効」を打ち消すためご注意ください。
    4. ゆがみ: グリフをわずかに歪ませることでグリッド配置を改善します。有効とするには、自動ヒンティングおよび「通常」ヒンティングモードにする必要があります。
    5. 拡大縮小なし: チェックした場合、グリフはフォント単位のままとなり、指定されたサイズを無視します。よってヒンティングは無効となります。
    上記のオプションは、デフォルトの状態ではすべて無効にされています。
  • インタープリター: GameMaker Studio 2 によるフォントレンダリングで利用される TrueType のバージョンを指定します。デフォルトでは TrueType v40 となります。
  • 丸め処理: サイズを固定小数点から整数へ変換するときの丸め処理を定めるオプションです。デフォルトでは「切り捨て」となります。

再生成ボタンをクリックすると、フォントテクスチャが再生成されます。既存のフォントに変更を反映させる場合や、新しく作成したフォントの見栄えをプレビューウィンドウで確認するときに実行する必要があります。なお、次のようにアセットブラウザからフォントを選択し、右クリックメニューを開いて複数のフォントを再生成することも可能です。

注意事項として、新しく作成したフォントのテクスチャをエディタで再生成していない場合、ゲームの実行またはテスト時にテクスチャが自動生成されますが、最初の一度しか生成されません。つまりテクスチャを一度生成してからは、フォントリソースに変更を加えるたびに再生成してゲーム内に表示を反映しなくてはならず、再生成するまでは「古い」フォントテクスチャが使用されることになります。

GameMaker Studio 2 において、生成されたフォントテクスチャは他のプロジェクトファイルとともに PNG ファイルとしてレンダリングされます。そのためプロジェクトが読み込まれたときに元のフォントが見つからない場合も、保存されているテクスチャを使用してゲーム内にテキストを正常に表示することができます。ただしフォントが見つからない場合、名前、フォントファミリー、テクスチャグループ（テクスチャグループエディタから変更）以外は変更できなくなります。


プレビューモードプレビューモード

プレビューウィンドウに選択中のフォントによるサンプルテキスト、あるいはフォントの生成済みレンダリングテクスチャ画像を表示するかを選ぶことができます。

 

プレビュープレビュー

プレビューウィンドウでは、サンプルテキストまたはテクスチャとして、フォントのプレビューを行うことができます。プレビューモードがサンプルに設定されている場合、テキストをプレビューウィンドウに入力することで表示をテストできます。ゲームプレビューに設定されている場合、フォントの生成済みレンダリングテクスチャ画像が表示されます。表示されない場合は、再生成ボタンをクリックしてテクスチャを生成しましょう。

テクスチャは自発的に操作しないと再生成されないため、テクスチャ画像にはサンプルとの差異が生じる場合があります。そのためフォントテクスチャが最初に生成されてから加えた変更も、テクスチャを再生成しなければゲーム内に引き継がれません。

 

範囲プレビュー範囲プレビュー

フォントに追加されている文字列範囲をリスト表示します。左側には範囲の値が表示され、右側にはその値に含まれているグリフが表示されます。
