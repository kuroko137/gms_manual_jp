シーケンスエディタ

シーケンスとは、動的なアニメーションを再生するための複合型アセットです。スプライト、インスタンス、サウンドといったアセットを追加でき、移動や色の変更、アニメーションの再生 / 停止といった動作をシーケンス内だけで行なわせることができます。シーケンスに追加されたアセットは特定のトラックに割り当てられます。トラックにはさまざまな属性（キーフレーム）を適用でき、それらは時間の経過によって変化させることができます。シーケンスの編集はすべてシーケンスエディタで行います。このエディタは次の 3 つの領域から構成されています。

  • キャンバス - シーケンスを構成するアセットを配置していく領域
  • トラックパネル - シーケンスにアセットトラックとパラメータートラックを追加 / 削除する領域
  • ドープシート - トラックのキーフレームを追加 / 削除 / 編集し、時間による動きを管理する領域

シーケンスの中にシーケンスを入れれば複雑なアニメーションやエフェクトを作成でき、さらにシーケンスはコードから参照して変更を加えられるため、ゲーム内でどのように見えるか、どのように扱われるかを可能な限りコントロールできます。シーケンスのコード面での詳細についてはシーケンス関数の該当項目を参照してください。

エディタについて詳しく説明する前に、シーケンスをどうやって作るのか、さらには文中で使われる用語についてお伝えしましょう。シーケンスエディタの仕組みが理解しやすくなるはずです。シーケンス作りは次のような手順からなります。

  • まずはアセットブラウザからシーケンスを作成し、シーケンスエディタで開きましょう。
  • 次にアセット（スプライト、オブジェクト、またはシーケンス）をエディタのキャンバスにドラッグしてください。
  • トラックパネルにアセットトラックが作成され、ドープシートにアセットキーが追加されます。アセットトラックとは、アセットが含まれているトラック名のことであり、アセットキーとは、アセットのアニメーション実行フレーム数を表しているバーのことです。
  • この段階では、キャンバスに置かれたアセットをシーケンス開始時の初期位置に動かし、さらに拡大縮小や回転といった初期変形を加えることになります。
  • 納得いくよう初期設定ができたら、パラメーターキー（もといキーフレーム）を記録しましょう。パラメーターキーとはスケール、回転、位置などの値を指す名前であり、アセットキーはこのパラメーターキーを、ドープシートタイムラインの任意地点に記録できます。
  • さらに再生ヘッド（ドープシートタイムラインでの現在の再生位置）を移動させ、アセットを動かしたり変形させて次のパラメーターキーを記録してください。
  • パラメーターキーは追加されたパラメータートラックに保存されます。パラメータートラックとは、トラックパネルのメインアセットトラックより下にできるサブセットトラックのことです。各パラメーターのキーデータはトラックエディタから編集可能です。
  • あとはアセットを移動、回転、拡大縮小させたり、足りないアセットがあれば追加するなど、上記の手順をシーケンスの最後まで繰り返すことになります。

以下はシーケンスエディタの各機能の概要です。ページの一番下からは、前述した 3 つのメイン機能についての詳しい解説ページに移動できます。

 

定規 / ガイド定規 / ガイド

キャンバスの縁にある定規は配置されたアセットの位置を示しており、キャンバスの中心ないしシーケンスのデフォルト原点である (0, 0) を起点としています。この定規をドラッグして引っぱると、縦・横方向のガイドがシーケンスに追加されます。アセットを動かしているときにガイドへ近づけると "スナップ" するため、これを利用して複数のアセットの位置を揃えることができます。アセットの配置中はさらにスマートガイドが表示され、アセット同士の距離や、シーケンスの境界線、中心点からの距離を示してくれます。

シーケンスの境界線、中心点のマーカー線もガイド扱いであり、定規から追加したガイドと同じ機能を持ちます。ただしこちらは編集できません。

他の設定に加えて、シーケンス環境設定からアセットがガイドにスナップするまでの距離を設定できます。

 

キャンバスビューキャンバスビュー

キャンバスビューではシーケンスキャンバスのプレビューを確認でき、シーケンスを構成するアセットを編集・配置できます。シーケンスにアセットを追加するには、アセットブラウザからキャンバスにアセットをドラッグしてください。するとトラックパネルに新しいトラックが作成され、タイムラインの再生ヘッド位置にアセットキーが追加されます。キャンバスにドラッグして追加できるのは次のアセットです。

  • スプライト
  • オブジェクト
  • サウンド
  • シーケンス

キャンバスに追加されたアセットは回転、変形、移動など複数の方法で操作でき、アセットに表示される境界ボックス、変形ギズモ、境界ボックスコントロールをどう扱うかで操作が変わります。詳しい説明はシーケンスキャンバスのページを参照してください。

メモ: シーケンスに追加されたサウンドアセットは、指定したサウンドリソースを再生するだけのサウンドエミッタとなります。

キャンバスビューは普通のワークスペースと同じように操作できます。中央クリックしながらドラッグで（ + の組み合わせでも可）キャンバスの移動、マウスホイールで垂直スクロール、 / キーを押しながらマウスホイールでズームができます。 / + "X" キーでアセットの切り取り、 / + "C" でコピー、 / + "V" で貼り付けも可能です。

 

キーフレーム操作部キーフレーム操作部

キーフレーム制御ボタンはドープシートの特定キーを追加 / 削除 / 編集するためのボタン群です。

  •  このボタンは「選択キー分割」ボタンです。ドープシートのパラメーターキーを 1 つ以上選択している状態でこのボタンをクリックすると、タイムラインの再生ヘッド位置でキーを 2 つに分割します（あらかじめ複数のフレームにまたがるようキーを引き伸ばしておく必要があります）。
  •  - このボタンは「選択キー削除」ボタンです。ドープシートのキーを 1 つ以上選択した状態でこのボタンをクリックすると、選択中のキーを削除します。
  •  - このボタンは「キーの新規記録」ボタンです。ドープシートのアセットキーを選択した状態でこのボタンをクリックすると、アセットトラックのパラメータートラックに新しいパラメーターキーを作成し、タイムラインの再生ヘッド位置にパラメーターキーのポイントが追加されます。
  •  - このボタンは「変更の自動記録」ボタンです。このボタンが押されているとキャンバス上のアセットに加えた変更が自動記録されるようになり、ドープシートの再生ヘッド位置に適切なパラメータートラックおよびパラメーターキーを追加します。たとえば、再生ヘッドを 0 フレーム目から 10 フレーム目に動かしてからキャンバス上のアセットを 100 ピクセル右に移動させると、パラメーターキーが 0 フレーム目（初期位置）と 10 フレーム目（再生ヘッド位置）に追加されます。この状態でシーケンスの「再生」ボタンを押すと、10 フレームかけてアセットが 100 ピクセル右に移動します。

シーケンスイベントシーケンスイベント

通常のオブジェクトと同じくシーケンスにはイベントを追加でき、実行させるコードを割り当てることができます。コードは引数を受け付けないスクリプト関数として割り当てられ、イベントが発生したタイミングで呼び出されます。

イベントの追加ボタンをクリックすると、次のウィンドウからイベントを追加できます。

以下は追加できるイベントの一覧です。

  • Create - シーケンスのインスタンスが最初に生成されたときに起きるイベントです。ルームエディタによって配置されたシーケンス内で最初に起きるイベントであり、ルームに入ったタイミングで発生します。
  • Destroy - シーケンスのインスタンスが破棄されたときに実行されるイベントです。以下の Clean Up イベントより先に実行されます。
  • Clean Up - シーケンスのインスタンスをルームから除去する全イベントの後に呼び出されるイベントです。つまりシーケンスの破棄、ルームの終了、ゲームの終了によって作動するイベントとなり、インスタンスがゲームから除去されたときに任意の処理を実行させる役割を持っています。シーケンスのインスタンスが layer_sequence_destroy() などで破棄された場合、このイベントは上記の Destroy イベントより後に呼び出されます。
  • Begin Step / Step / End Step - シーケンスが存在している間、ゲーム内での 1 ステップ（フレーム）ごとにチェックされるイベントが Step イベントであり、Begin 、Step 、End という 3 つの段階に分けられています。ほとんどの場合は普通の Step イベントで問題ありませんが、どのコードをどのタイミングで実行させるかをもう少しコントロールしたい場合もあるでしょう。そのために用意されているイベントが Begin と End イベントです。この二つのイベントはゲーム内での 1 ステップ（フレーム）ごとに、必ず決められた順番で作動します。注意！シーケンスが一時停止されている間、Step イベントは作動しません。シーケンスが再開された場合、再生を再び始めた直後のフレームで作動します。また、再生ヘッドの方向によってイベントの順番が変わることはありません。シーケンスが逆再生されている場合でも、Step イベントは Begin 、Step 、End の順に実行されます。
  • Async Event: オブジェクトの Asynchronous System Event に相当するイベントです。注意！シーケンスが一時停止されている間、このイベントは作動しません。

各イベントには、イベントの作動時に呼び出す関数を 1 つ割り当てることができます。関数の割り当てはイベントの入力ボックスから行ない、矢印ボタンをクリックするとスクリプトエディタを開いて関数を編集することもできます。また、一番下にある新規作成ボタンをクリックすると「定型」関数入りのスクリプトリソースが作成され、すべてのイベントに割り当てられます。関数の名前は自由に変更できますが、すでに定義されている名前は使用しないようにしてください。不要となった関数は定義を削除するといいでしょう。なお、シーケンスイベントで使用される関数は引数を受け取らないため注意してください。

シーケンスイベントはコードから追加・編集することも可能です。詳しくはこちらを参照してください。

 

トラックパネルトラックパネル

トラックパネルにはシーケンスに追加したアセットがトラックとして一覧表示されており、各トラックにはアセットの変更パラメーターを記録するサブトラックが付いています。アセットトラックはドラッグすることで他のトラックの上や下に移動でき（割り当て中のパラメーターもまとめて移動します）、 / + クリックで複数選択ができます。

トラックを作成するには、アセットブラウザからアセット（スプライト、オブジェクト、サウンド、シーケンスのいずれか）をシーケンスキャンバスにドラッグしましょう。または一番下にあるアイコンをクリックするとアセットエクスプローラーが開きますので、そちらからアセットを選んで新しいトラックを作成することもできます。トラックが作成されると、以下のようにドープシートタイムラインの再生ヘッド位置にアセットが追加されます。

トラックエディタにはこの他にも多くの機能があり、以下のページで詳しく説明されています。

  • トラックパネル

 

キャンバスのツールボックスキャンバスのツールボックス

シーケンスエディタのツールボックスでは、表示の切り替えや、シーケンスの編集方法を設定できます。

  • キャンバスのグリッド表示切替: シーケンスエディタのキャンバスに表示されるグリッドをオン / オフにします。このグリッドはプレビューキャンバスを分割するために GameMaker Studio 2 側で描画されるものであり、デフォルトのサイズは 32 x 32 ピクセルとなっています。グリッドのメニューアイコンをクリックすると次のオプションが開きます。
    オプションを設定することで、グリッドの色や透明度、X 軸と Y 軸のセル幅を変更できます。また、グリッドのスナップを有効 / 無効にするオプションもあります（デフォルトでは無効）。グリッドの表示はキーボードショートカット "G" で、グリッドのスナップは + "G" で切り替えることも可能です。
  • キャンバスのズーム操作部 : 左のボタンからキャンバスのズーム倍率を変更できます。ズームイン、ズームアウトのほかに、ボタンをクリックすれば編集中のアセットが等倍になるようキャンバスをリセットできます。また、ウィンドウにフィットボタンをクリックすると、キャンバス全体をエディタのワークスペース内に収めることができます（フィットするようズームイン / ズームアウトされます）。ズーム倍率は / を押しながらマウスホイールでも変更可能です。また、 / + でキャンバスを等倍表示にできます。
  • ギズモ切替 : クリックすると、キャンバスに表示されるギズモを有効 / 無効にできます。有効にした場合、キャンバスと各アセットに対応するギズモが表示されます。無効にすると表示されなくなり、シーケンスをすっきり見通せるようになります。デフォルトでは有効となります。また、設定ボタンをクリックしてメニューを開くと、ウィジェットをすべて無効にすることなく個別にフィルター表示できます。フィルター設定の一覧および機能についての説明はこちらを参照してください。
  • ギズモ変形モードの切替 : シーケンスに追加されたアセットの原点には、このように変形ギズモが表示されます。
    変形ギズモを操作することで、アセットのスケールや回転、位置や原点を設定できますが、同時に表示されるギズモは 1 種類のみです。ボタンをクリックするとギズモのウィジェットが切り替わり、設定メニューをクリックすれば操作タイプを直接選択できます。
  • キャンバスフレーム切替 : このボタンをクリックすると、キャンバスフレームのオン / オフを切り替えることができます（デフォルトではオン）。キャンバスフレームはシーケンスキャンバスのどこに要素が置かれているかを判断するためのガイドであり、フレームの外にある要素はレンダリングされません。このボタンには次のような設定メニューがあります。
    メニューからはキャンバスフレームの切り替え、キャンバスのサイズを設定できます。また、参考画像の設定も可能で、パソコンにある画像を選んでシーケンスの参考用背景にすることができます（プロジェクトにはインポートされません）。参考画像はシーケンスエディタ内でのみ表示され、プロジェクトで使用されるシーケンスにはレンダリングされません。この画像を追加すると、参考画像の透明度およびキャンバスフレームに対する位置オフセットを設定できるようになります。

 

シーケンスツールシーケンスツール

以下のバーからシーケンスの再生を制御できます。

  • 再生速度: シーケンスの再生速度です。1 秒ごとのフレーム（ステップ）数で表されます。
  • 再生ヘッド位置: 再生ヘッドの現在位置です。時間 (h:m:s) またはフレーム (f) 単位で表示されます。
  • 時間の表示設定: シーケンスの時間単位を時間 / 分 / 秒 (h:m:s) で表示するかフレーム数 (f) で表示するかを設定できます。
  • ブロードキャストメッセージ: クリックすると、再生ヘッド位置にブロードキャストメッセージを追加します（詳しくはこちらを参照）。
  • モーメントを追加: クリックすると、再生ヘッド位置にシーケンスモーメントを追加します（下に詳細あり）。
  • カーブエディタを開く: 選択トラックのカーブビューとキーフレームビューを切り替えるボタンです。パラメータートラックにのみ適用できるオプションで、有効にするとパラメータートラックがキーフレームを使用しているか、アニメーションカーブを使用しているかによって追加のオプションが表示されます。詳しくはアニメーションカーブの使い方セクションを参照してください。
  • 再生操作部: これらのボタンをクリックすると、シーケンスのプレビューを開始 / 停止したり、再生ヘッドを開始位置 / 終了位置まで移動できます。また、シーケンスをループ再生させるかどうか、「ピンポン」か繰り返しのどちらでループさせるかを設定可能です。ここでの設定は、プロジェクトをコンパイルして実行した際のシーケンス再生方法に影響するためご注意ください。
  • シーケンス長: シーケンスの最大長がフレーム (f) または時間 (m:h:s) 単位で表示されます。クリップ領域がこれより狭く設定されている場合は、表示されている長さまで再生されません。

上にあるように、シーケンスにはモーメントを追加できます。シーケンスにおけるモーメントとは、シーケンスが関数を呼び出すタイムライン上の位置のことです。この関数は引数をもたないスクリプト関数でなくてはならず、タイムラインがモーメント（フレーム）まで進んだタイミングで呼び出されます。モーメントを設定するには、再生ヘッドを目的の位置まで移動させてからモーメントを追加ボタンをクリックしてください。ダイアログが開きますので、呼び出したい関数の名前を入力しましょう。

モーメントの関数が追加されている場合、ボタンをクリックすると指定された関数のスクリプトに移動できます。また、ボタンをクリックすることで新しいスクリプトリソースを空の関数で作成して編集できます。ボタンをクリックした場合は、モーメントおよび関数の呼び出しを削除します。

シーケンスのモーメントはコードからも編集・追加が可能です。詳しくはこちらを参照してください。

 

ドープシートドープシート

ドープシートは各トラックのアセットキーおよびパラメーターキーを表示する領域です。キーをクリックして左右にドラッグするとドープシートタイムラインにおけるキー配置が変わり、アセットキーの右端をクリックすると長さを変更できます。スプライトやオブジェクトの場合、長さを実際のアニメーションよりも長く設定すると、余分なフレーム数アニメーションがループするため注意してください。ドープシートでは、シーケンスのクリップ領域の始点と終点を設定できます。また、任意の位置をクリックすると再生ヘッドがその位置まで移動します。ドープシートの詳しい使い方については以下のページを参照してください。

  • ドープシートの使い方

 

向きを変更向きを変更

向きを変更ボタンをクリックすると、シーケンスエディタの各部分をどのように表示するか変更できます。デフォルトではキャンバスビューが上、トラックエディタとドープシートが下に配置されていますが、このボタンをクリックすると順番に異なるレイアウトへと切り替わります。また、設定ボタンをクリックするとメニューが開き、レイアウトを直接指定することができます。利用できるレイアウトは次の通りです。

ウィンドウを分割

シーケンスエディタが 2 つのウィンドウに分かれます。IDE のメインウィンドウにはキャンバスビューが全体表示され、新しいウィンドウにはドープシートとトラックエディタが表示されます。ボタンをもう一度クリックするとレイアウトが切り替わり、新しいウィンドウが閉じられます。
上にスナップ

トラックエディタとドープシートが IDE の上側にスナップします。
右にスナップ

トラックエディタとドープシートが IDE の右側にスナップします。
下にスナップ

トラックエディタとドープシートが IDE の下側にスナップします（デフォルトのビュー表示）。
左にスナップ

トラックエディタとドープシートが IDE の左側にスナップします。

シーケンスエディタのレイアウトは、リセットボタンをクリックすることでいつでも元に戻せます。レイアウトをリセットすると、エディタがデフォルトの状態に戻り、閉じられていた部分も開き直されます。
