オブジェクト変数

変数ボタンをクリックすると、このようにオブジェクト変数ウィンドウが開きます。ここではオブジェクトの Create イベントより前に作られる変数を追加できます。

ウィンドウ内の追加ボタンをクリックすると、新しい変数が一覧に追加されます。追加した変数には名前をつけたり、型の切り替え、設定値の変更が可能です。

変数の名前は英字から始まる名前にしなくてはなりません。また、利用できるのは 64 字までの英数字および'_'記号のみとなります。fish 、foo_bar 、num1 などは有効な変数となりますが、6fish 、foo bar 、*num などは無効な変数となります。名前をつけたら、以下のうちから型を選ばなくてはなりません。オブジェクト変数は、親と子のインスタンスを扱うにあたって特に役立つ機能です。親オブジェクトに変数を定義すれば、子オブジェクトでは値を変更するだけで済みます（詳しくは下記のオブジェクト変数と子オブジェクトをお読みください）。また、ルームエディタにインスタンスを追加した場合にも（レイヤー > インスタンスレイヤーの段落を参照）、ルームに配置された個々のインスタンスの変数を変更することができます。

作成できる変数の型は以下のとおりです。

  • Real （実数）: 実数はそのすべてが整数ではない値であり、正の値にも負の値にもなります。たとえば 124.5 、45639.566546456 、0.9 、-45.5 などは実数となります。すべての実数は 64bit の浮動小数点の値 （ Double 型）として保存されます。実数（および整数）のデータ形式では値の範囲を設定でき、設定ボタンをクリックすると変更できます。チェックすると開始値と終了値の入力が有効となり、オブジェクト変数のウィンドウには固定値ではなく、範囲から値を決めるスライダーが表示されます。
  • Integer （整数）: 整数はそのすべてが数値の値であり、正の値にも負の値にもなります。たとえば 30004 、19 、0 、-300 などは整数です。実数の説明にあるように、整数にも出力値の範囲を設定できます。
  • 文字列: ダブルクォーテーション記号で囲まれたものは文字列となります。たとえば "fish" 、"Hello World" 、"12345" などは文字列です。
  • 真偽値 （ Boolean ）: 真偽値とは true （真）か false （偽）のどちらかになる値です。オブジェクトエディタの変数ウィンドウにはチェックボックスが表示され、チェックを入れると true に、外すと false になります。
  • 式 （Expression ）: 式とは通常の数、変数、文字列、あるいは GML 関数を含み、複数の演算式を利用できる数式文のことです。たとえば sqrt(85 * 6) + 5.5 などは式となります。
  • リソース: リソースは、アセットブラウザにあるいずれかのリソースとなります。リソース型にすると、アセットエクスプローラーを開くボタンをクリックしてアセットエクスプローラーを開けるようになり、そこから使いたいリソースを選択できます。
  • リスト: リスト型を選ぶと、複数の値（文字列、実数、式などが利用可）を作成し、その中から選択した 1 つまたは複数の値を返すことができます。リストに含まれる項目を定義するには、まずはボタンをクリックして設定から項目を追加しなくてはなりません。
    メインウィンドウのドロップダウンメニューから選んだ項目が変数の初期値となります。
    なお、リストの設定にある複数選択をチェックした場合、選んだすべての項目を含めた配列となります。
  • カラー: カラー型は変数に保存されるカラーの値を設定できます。実数 （0 から 16777216 までの値で、255 のアルファ値も含まれます）、16 進数値（$RRGGBBAA 形式）を入力できるほか、色見本をクリックするとカラーピッカーを開いて色を決めることができます。

 

 

オブジェクト変数と子オブジェクト

オブジェクト変数の大きな特徴は、アセットブラウザの子オブジェクトに継承されるということであり（詳しくは親オブジェクトの該当項目を参照してください）、任意の変数を上書きして変更することも可能です。

子オブジェクトの親となるオブジェクトリソースに変数が定義されている場合、オブジェクト変数ウィンドウはこのように表示されます。

上の画像は親オブジェクトで、4 つの変数が表示されています。下の画像にも同じ変数はあるものの、薄く表示されており、親から継承アイコンが付いているため継承された変数であることが分かります。これらは変数を上書きボタンをクリックすれば編集可能となり、値や範囲スライダーを調整したり、リストの別の項目を選んだりすることができます。なお、親の変数においては定義値だけを変更でき、変数の名前、設定は変更できません。また、子オブジェクトには新しいオブジェクト変数も追加可能です（画像の例では、子オブジェクトにだけ "Character Type" という変数があります）。

継承されたオブジェクト変数を編集した場合、削除ボタンをクリックすると編集内容を消去できますが、変数そのものは消すことができません。変数はそのまま親から継承され、値も親の初期値に戻ります。変数を完全に削除したいなら、親オブジェクトから削除しなくてはなりません。
