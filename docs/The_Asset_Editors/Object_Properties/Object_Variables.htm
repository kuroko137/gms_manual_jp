オブジェクト変数

変数ボタンをクリックすると、このようにオブジェクト変数ウィンドウが開きます。ここではオブジェクトの Create イベントの前に生じる変数を追加できます。

ウィンドウの追加ボタンをクリックすると、新しい変数が一覧に追加されます。追加した変数には名前をつけたり、型の割り当て、設定値の変更が可能です。

変数の名前は文字から始まる名前にしなくてはなりません。また、利用できるのは 64 字までの英数字または'_'記号のみとなります。fish 、foo_bar 、num1 などは有効な変数となりますが、6fish 、foo bar 、*num などは無効な変数となります。名前をつけたら、以下のうちから型を選ばなくてはなりません。オブジェクト変数は、親と子のインスタンスを扱うにあたって特に役立つ機能です。親オブジェクトに変数を定義すれば、子オブジェクトでは値を変更するだけで済みます（詳しくは下記のオブジェクト変数と子オブジェクトをお読みください）。また、ルームエディタにインスタンスを追加した場合も同じように（レイヤー > インスタンスレイヤーの段落を参照）、ルームに配置された個々のインスタンスの変数を変更することができます。

作成できる変数の型は以下のとおりです。

  • Real （実数）: 実数とはそのものが整数でない数値であり、正の値にも負の値にもなります。たとえば 124.5 、45639.566546456 、0.9 、-45.5 などが実数の例となります。実数はいずれも 64bit の浮動小数点の値 （Double 型）として保存されます。実数（および整数）のデータ型には値の範囲を設定できるオプションがあり、オプションボタンをクリックすると利用できます。チェックを入れると開始値と終了値を入力できるようになり、オブジェクト変数のウィンドウには固定値ではなく、範囲から値を固定できるスライダーが表示されます。
  • Integer （整数）: 整数とはそのものが数値からなり、正の値にも負の値にもなる 30004 、19 、0 、-300 のような値のことです。上記の実数で説明したように、整数の出力値にも範囲を設定できます。
  • 文字列: ダブルクォーテーション記号で囲むと文字列となります。たとえば "fish" 、"Hello World" 、"12345" などです。
  • 真偽値 （Boolean ）: 真偽値とは true （真）か false （偽）のどちらかにだけできる値です。オブジェクトエディタの変数ウィンドウにはチェックボックスとして表示され、チェックすると true に、チェックを外すと false になります。
  • 式 （Expression ）: 式とは普通の数、変数、文字列、あるいは GML 関数を含み、複数の演算式を利用できる数式文のことです。たとえば sqrt(85 * 6) + 5.5 などは式となります。
  • リソース: リソースは、アセットブラウザで設定したいずれかのリソースとなります。リソース型にしたときは、アセットエクスプローラーを開くボタンをクリックするとアセットエクスプローラーが開き、使いたいリソースを選択できます。
  • リスト: リスト型を選んだ場合は、選択対象となる複数の値（文字列、実数、式などが利用可能です）を作成し、その中から指定した 1 つまたは複数の値を返すことができます。リストに含まれる項目を定義するには、まずはボタンをクリックしてオプションから項目を追加しなくてはなりません。
    メインウィンドウのドロップダウンメニューから指定した項目が変数のデフォルト値となります。
    なお、リストのオプションにある複数選択をチェックした場合、選択したすべての項目を含めた配列となります。
  • カラー: カラー型は変数に保存されるカラーの値を設定できます。実数 （0 から 16777216 までの値で、255 のアルファ値も含まれます）、16 進数値（$RRGGBBAA 形式）を入力できるほか、色見本をクリックするとカラーピッカーを開いて色を決めることができます。

 

 

オブジェクト変数と子オブジェクト

オブジェクト変数の大きな特徴は、アセットブラウザの子オブジェクトに継承されるということであり（詳しくは親オブジェクトの該当項目を参照してください）、継承された変数のうち、指定したものだけを上書きあるいは変更できます。

作成した子オブジェクトの親となるオブジェクトリソースに変数が定義されている場合、オブジェクト変数ウィンドウはこのように表示されます。

上の画像に表示されているのは親オブジェクトの 4 つの変数で、下の画像でも確認できますが薄く表示されており、継承された変数であることを示している親から継承アイコンがついています。継承された変数は変数を上書きボタンをクリックすると編集可能で、範囲スライダーを調整したり、リストの別の項目を選ぶといったことができます。なお、親の変数では定義値だけを変更でき、変数の名前やオプションなどは変更できません。また、子オブジェクトにも新しいオブジェクト変数を追加することは可能です（画像の例では、子オブジェクトにだけ "Character Type" という変数があります）。

継承されたオブジェクト編集を編集した場合、削除ボタンをクリックすると編集内容を消去できますが、変数そのものは消せません。変数は引き続き親から継承され、親のデフォルト値がまた使用されるようになります。変数を完全に消したいなら、親オブジェクトから変数を削除する必要があります。
