In App Purchase

このイベントは、マーケットプレイスにある各ターゲットプラットフォーム向けの In App Purchase (IAP) 拡張機能によってのみ作動します。作動するとこのイベントだけで使える DS マップが生成され、async_load という専用変数に格納されます。DS マップにはイベントを作動させた拡張関数に応じたキー / 値ペアが含まれますが、イベントのタイプを識別するための "id" キーはどの関数でも追加されます。戻り値やこのイベントを発生させる関数についての詳細は、拡張機能に付属している PDF マニュアルを参照してください。
