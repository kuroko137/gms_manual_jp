コードスニペット

コードを編集するにあたって便利に扱えるツールがコードスニペットです。スクリプトエディタでは F4 を押すとコードスニペット画面が表示され、よく使われるコードメソッドを一覧から選択できます。

マウスを使うか、リストの右側に表示されているホットキーを押すとスニペットが選択され、編集中のコードに追加できます。

自作したコードスニペットを定義することも可能です。そうするにはまず、以下のディレクトリに "snippets.txt" というファイルを作成しなければなりません。

  • Windows: %ProgramData%\GameMakerStudio2\User\
  • macOS: /Users/Shared/GameMakerStudio2/User/

このフォルダは GameMaker Studio 2 がアップデートされても影響を受けず、編集したファイルもそのままとなります（ただしインストール先のディレクトリにある元のスニペットファイルは初期化されます）。

ユーザーディレクトリにファイルを作成したらテキストエディタで開き、以下の規則に従って編集しましょう。

  • スニペットはそれぞれ別々の行に追加する（空白行を入れることは不可）
  • スニペットの先頭にはホットキー（直後に "-" ）が入り、次はメニューで表示されるスニペット名（直後にクーロン ":" ）が入ります。

I - インスタンス生成:

クーロンの直後にコードのスニペットを追加します。

I - インスタンス生成:instance_create_layer(x, y, |layer|, object);

追加するコードも一定の形式に合わせなくてはなりません。

  • コードは 1 行以内に収める
  • 新しい行は "$" で区切る
  • 最初に選択させたいテキストは "|" 記号で囲む（スニペット追加時にカーソルがそのテキストを編集できるよう選択）

上記の規則に従ってどのように設定されているかは、元のスニペットファイルに含まれている見本を参照してください。元ファイルは GameMaker Studio 2 のインストールディレクトリ内にあります。

  • \GameMaker Studio 2\TextEditor\snippets.txt
