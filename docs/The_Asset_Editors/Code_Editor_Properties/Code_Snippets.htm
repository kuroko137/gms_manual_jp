コードスニペット

コードを編集する際、気軽に使えて便利なものがコードスニペットです。スクリプトエディタでは F4 を押すとコードスニペットウィンドウが開き、よく使われるコード文が一覧表示されます。

>このウィンドウではマウス、またはリストの右に表示されているホットキーからコードスニペット（再利用できる短いコード）を選択でき、選択したスニペットは編集中のコードに追加されます。

また、自分だけのコードスニペットを定義することも可能です。まずは以下のディレクトリに "snippets.txt" というファイルを作成しましょう。

  • Windows: %ProgramData%\GameMakerStudio2\User\
  • macOS: /Users/Shared/GameMakerStudio2/User/

このフォルダは GameMaker Studio 2 のアップデートによる影響を受けないため、変更したファイルもそのままとなります（ただしインストール先のディレクトリにある基本スニペットファイルは元に戻ります）。

ユーザーディレクトリにファイルを作成したらテキストエディタで開き、以下の規則どおりに編集しましょう。

  • スニペットはすべて別の行に分ける（空白行は含めることができません）
  • スニペットの先頭はホットキー（直後に "-" ）、続いてメニューでの表示名（直後に ":" ）

I - インスタンス生成:

最後にコードスニペットを追加

I - インスタンス生成:instance_create_layer(x, y, |layer|, object);

追加するコードに関しても、以下の形式に合わせなくてはなりません。

  • コード全体は 1 行以内に収める
  • 新しい行として扱いたい箇所には "$" を挿入する
  • 自動選択させたいテキストは "|" 記号で囲む（スニペット追加時にカーソルが囲まれたテキストを選択）

どのように設定すればいいのか見本を見たい場合は、基本スニペットファイルを開きましょう。このファイルは GameMaker Studio 2 のインストールディレクトリ内にあります。

  • \GameMaker Studio 2\TextEditor\snippets.txt
