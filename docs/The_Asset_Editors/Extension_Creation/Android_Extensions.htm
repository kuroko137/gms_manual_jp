Android 拡張機能

Android の拡張機能は 2 つの手順から作成します。1 つ目は拡張機能および必要ファイル等の追加、2 つ目は関数およびマクロ / 定数の作成です。関数と定数はプレースホルダによってグループ化して追加されるため、あらかじめプレースホルダのファイルを追加してから関数とマクロを定義しなくてはなりません。残りのファイルを追加するには、まずエディタの追加プラットフォーム領域にある Android にチェックを入れてプロパティを開いてください。

以下はプロパティの詳細です。

  • クラス名: 拡張機能には複数のクラスを用意でき、クラスごとに独自の関数、定数を追加できます。他のクラスと用途が区別できる名前をつけましょう。
  • Android 権限: 拡張機能に権限を追加できます。必要となる権限は拡張機能の用途によって異なるため、Google の Android プラットフォーム用ドキュメントに目を通しておくといいでしょう。サードパーティの SDK を利用しているのであれば、SDK に同梱されているドキュメントを確認しましょう。権限を追加するには、まずボタンをクリックしてプレースホルダ権限を追加してください。追加した権限はゆっくりダブルクリックすると編集できますので、必要な権限に変更しましょう。権限はボタンで削除可能です。
  • Gradle 依存関係へ挿入: Gradle ビルド依存関係に挿入（追加）しておきたいコードを入力します。
  • AndroidManifest.xml マニフェストへ挿入: ゲームのテストビルドおよび最終ビルド時に、Android マニフェスト XML ファイルに挿入（追加）されるコードを入力します。ゲームをストアに申請する際、必ずこの設定（および権限）を見直すようにしてください。設定ミスがあれば申請失敗となるでしょう。
  • AndroidManifest.xml アプリケーションへ挿入: ゲームのテストビルドおよび最終ビルド時に、Android マニフェスト XML ファイルのアプリケーションヘッダー以下に挿入（追加）されるコードを入力します。ゲームをストアに申請する際、必ずこの設定（および権限）を見直すようにしてください。設定ミスがあれば申請失敗となるでしょう。
  • AndroidManifest.xml RunnerActivity へ挿入: ゲームのテストビルドおよび最終ビルド時に、Android マニフェスト XML ファイルの RunnerActivity ヘッダー以下に挿入（追加）されるコードを入力します。ゲームをストアに申請する際、必ずこの設定（および権限）を見直すようにしてください。設定ミスがあれば申請失敗となるでしょう。

正しく設定できたら、拡張機能のパッケージ動作に必要なファイルを追加しましょう。一番下にある「 SDK を追加」または「ソースを追加」ボタンをクリックすれば、指定したファイルを追加できます。ファイルは拡張機能の AndroidSource ディレクトリに保存され、拡張機能を右クリックして「拡張機能のディレクトリを開く」を選ぶと保存先を開くことができます。

ndroid 拡張機能のより詳しい作成方法については、YoYo Games ナレッジベースを参照してください。
