Android の拡張機能

Android の拡張機能を作成するには 2 つの手順を踏まなければなりません。1 つ目の手順は拡張機能を必要なファイル等の追加、2 つ目の手順はマクロ / 定数、関数の作成です。関数と定数はグループ化するためにプレースホルダファイルを通して追加されるため、前述したようにプレースホルダを追加してから関数とマクロを定義しなければなりません。残りのファイルを追加するには、まずエディタの追加プラットフォーム部分にある Android をチェックする必要があります。

こちらの詳細は次のとおりです。

  • クラス名: 拡張機能には複数のクラスを与えることができ、クラスごとに専用の関数、定数を用意できます。用途がわかるような名前にしましょう。
  • Android 権限: 拡張機能に必要な権限を追加できます。必要となる権限は拡張機能の用途によって異なるため、Google の Android プラットフォーム向けドキュメントを確認しておくべきでしょう。サードパーティの SDK を利用している場合は、SDK に同梱されているドキュメントを確認しましょう。新たに権限を追加するには、ボタンをクリックしてプレースホルダ権限を追加したのち、ゆっくりダブルクリックして必要な権限になるよう編集してください。権限はボタンで削除できます。
  • Gradle 依存関係へ挿入: Gradle ビルド依存関係へ挿入（追加）する必要のあるコードを入力できます。
  • AndroidManifest.xml マニフェストへ挿入: ゲームのテストビルドおよび最終ビルド時に、Android マニフェスト XML ファイルへ挿入（追加）されるコードを入力します。ゲームをストアに申請する前に、必ずこの設定（および権限）を見直すようにしてください。設定に間違いがあると申請に失敗してしまうでしょう。
  • AndroidManifest.xml アプリケーションへ挿入: ゲームのテストビルドおよび最終ビルド時に、Android マニフェスト XML ファイルのアプリケーションヘッダー以下へ挿入（追加）されるコードを入力します。ゲームをストアに申請する前に、必ずこの設定（および権限）を見直すようにしてください。設定に間違いがあると申請に失敗してしまうでしょう。
  • AndroidManifest.xml RunnerActivity へ挿入: ゲームのテストビルドおよび最終ビルド時に、Android マニフェスト XML ファイルの RunnerActivity ヘッダー以下へ挿入（追加）されるコードを入力します。ゲームをストアに申請する前に、必ずこの設定（および権限）を見直すようにしてください。設定に間違いがあると申請に失敗してしまうでしょう。

正しく設定できたら、拡張機能のパッケージ動作に必要なファイルを追加しましょう。一番下にある「 SDK を追加」または「ソースを追加」ボタンをクリックし、ファイルを選んでください。追加されたファイルは拡張機能の AndroidSource ディレクトリに保存されます。拡張機能を右クリックし、「拡張機能のディレクトリを開く」を選べば保存先を開くことができます。

Android の拡張機能の作成方法については、YoYo Games ナレッジベースにより詳しい情報があります。
