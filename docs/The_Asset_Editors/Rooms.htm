ルームエディタ

ルームエディタでは、ゲームのルームを作成できます。どんなゲームでも実行するには 1 つ以上のルームが必要となり、ルームにはインスタンスやスプライト、タイル、背景、パスといったアセットを置くことができます。それぞれのアセットは専用のレイヤーに置かれることになり、レイヤーエディタからはレイヤーの順番を並び替えることが可能です。ルームエディタは複雑なツールため、ここでは一番大事な機能について簡単に紹介しましょう。詳しい領域については下側に並べられているリンク先を見てください。

ルームリソースを作成するには、アセットブラウザを右クリックしてメニューを開きます。メニューからは子ルーム（詳しくは継承関係のページを参照）を作成したり、作成したルームをエディタから編集できるほか、グループを追加してルームを整理したり、ルームの名前変更や削除ができます。なお、ルームの順番、継承関係についてはルームマネージャーから変更しなくてはなりません。このツールはアセットブラウザ右上のメニューから開けます。

ルームエディタは 1 つのワークスペースであるため、タブをクリックしてメインウィンドウの外にドラッグすれば新しいウィンドウとして分離できます（別のディスプレイまで動かすことも可能です）。分離したウィンドウは、タブをクリックして IDE の上部にドラッグすればメインウィンドウに戻せます。

ルームエディタには簡単に操作できるインターフェースが備わっています。インターフェースは複数の領域に分かれており、そのうちルームプロパティ、レイヤーエディタ、レイヤープロパティの領域はエディタにドックされていますが、ワークスペースの外にドラッグすればドックを解除できます。元に戻したいときは、ワークスペースの上下左右の端にドラッグしてください。

以下の節は、ルームエディタの各領域についての概要です。

定規 / ガイド定規 / ガイド

キャンバスの端にあるものは定規で、アセットが置かれている位置を示しています。この定規はキャンバスの中央であり、ルームの原点となる (0, 0) を開始点としています。また、定規を引っぱるようにドラッグするとガイドを追加できます。アセットをガイドのそばまで動かすと「スナップ」するため、ガイドを追加すれば複数のアセットを正確に配置できるようになります。アセットの配置中はスマートガイドも表示され、アセット同士の距離や、ルームの境界線、中心点からの距離を示してくれます。

ルームエディタの環境設定では他の設定に加えて、アセットがガイドにスナップするまでの距離を設定できます。


レイヤーエディタレイヤーエディタ

ルームエディタでは、ルーム内のレイヤーにアセットが配置されます。レイヤーにはそれぞれ個別の「深度」があり、レイヤーウィンドウの下側に並んでいるレイヤーは、上側のレイヤーより奥に描画されます。

重要！ レイヤー深度には-16000 から 16000 までの上限値があり、上限外のレイヤーに置かれたインスタンスは描画されません。ただしイベントはすべて通常どおりに実行されます。

レイヤーを作成するには各ボタンをクリックします。作成されるレイヤーはボタンと以下のように対応しています。

インスタンス ゲーム内で利用するインスタンスを置くレイヤーです。
タイルマップ タイルセットのタイルマップを使ってタイルを置くレイヤーです。
パス ルームに追加したパスが表示されるレイヤーです。
アセット スプライトやシーケンスといった視覚的アセットを追加するレイヤーです。
背景 背景を定義するレイヤーです。基本的には単一色、またはスプライト画像となり、背景はルーム全体に描画されます。

なお、フォルダボタンをクリックするとレイヤーフォルダを作成し、選択したレイヤーをまとめることができます。削除ボタンをクリックした場合、選択したレイヤーをすべて削除します。レイヤーの種類についてはこちらで詳しく取り上げられています。

レイヤーエディタでは継承の切り替えもでき、この設定はレイヤーの順番、表示に影響します。表示については横にある目アイコンをクリックして直接変更できるほか、一番上のアイコンから全レイヤーをまとめて変更可能です。南京錠アイコンをクリックした場合はレイヤーを「ロック」し、編集を禁止できます。なお、非表示フラグが付けられたレイヤーはゲーム実行時に表示されなくなります。

 

レイヤープロパティレイヤープロパティ

ルームのレイヤープロパティウィンドウは、レイヤーエディタの選択レイヤーに合わせて内容が変化します。

各ウィンドウのプロパティ、リストは選択しているレイヤーの種類と結びついており、ここではレイヤーの表示方法や、配置されているアセットに関する基本データを編集できます。詳しくはこちらを参照してください。


ルームプロパティルームプロパティ

この部分では、ルームのサイズ、物理の使用、カメラおよびビューポートの利用といったルームの基礎的なプロパティを設定できます。詳しい内容についてはこちらで取り上げられています。


レイヤーツールボックスレイヤーツールボックス

一部のレイヤーでは、IDE 上部のレイヤーツールボックスに専用のツールが追加されます（タイルレイヤーやパスレイヤーなどの場合）。追加されるツールは編集しているレイヤーの種類によって変わり、こちらのレイヤー解説ページで紹介されています。


ルームツールボックスルームツールボックス

ルームエディタのメインキャンバス上部にはこれらのボタンがあり、キャンバスの表示内容を設定できます。

  •  キャンバスのグリッド表示切替: ルームエディタのキャンバスに表示されるグリッドをオン / オフにします。このグリッドはプレビューキャンバスを分割するために GameMaker Studio 2 側で描画されるものであり、デフォルトのサイズは 32 x 32 ピクセルとなっています。グリッドのメニューアイコンをクリックすると次のオプションが開きます。オプションを設定することで、グリッドの色や透明度、X 軸と Y 軸のセル幅を変更できます。また、グリッドのスナップを有効 / 無効にするオプションもあります（デフォルトでは無効）。グリッドの表示はキーボードショートカット "G" で、グリッドのスナップは + "G" で切り替えることも可能です。
  •  キャンバスのズーム操作部: 左のボタンからキャンバスのズーム倍率を変更できます。ズームイン、ズームアウトのほかに、ボタンをクリックすれば編集中のアセットが等倍になるようキャンバスをリセットできます。また、ウィンドウにフィットボタンをクリックすると、キャンバス全体をエディタのワークスペース内に収めることができます（フィットするようズームイン / ズームアウトされます）。ズーム倍率は / を押しながらマウスホイールでも変更可能です。また、 / + でルームのキャンバスを等倍表示にできます。
  • ビューを表示: クリックするとビューボックスの表示 / 非表示を切り替えます。表示にした場合、有効となっているルームの各ビュー領域がハイライト表示されます。
  • アニメーションを再生: スプライトやシーケンス、アニメーションタイルを追加していると、プロジェクトをコンパイルせずに実際の見た目を確かめたくなるかもしれません。このボタンをクリックするとすべてのアニメーションが再生され、もう一度クリックするとアニメーションが停止します。
  • すべてのレイヤーから選択: キャンバスに置かれたアセットをクリックしても、デフォルトでは編集中のレイヤーにあるものしか選択できません。このオプションを有効にした場合、すべてのアセットをクリックして選択できるようになり、レイヤーも選択したアセットのものに切り替わります。なお、キーボードショートカット "P" からも一時的に有効にできます（ "P" を押している間は有効に、離すと無効になります）。

     

ルームキャンバスルームキャンバス

The center of the room editor window is taken up with the area where all the actual editing takes place. Here is where you'll be placing your instances and assets, drawing your tiles, or positioning your paths. You can zoom in and out using the mouse wheel or the room controls at the top, and you can pan around by holding the middle mouse button down and moving the mouse around or using + while moving.

When creating asset layers or instance layers, you can place the asset or instance by simply dragging it from the Asset Browser and then dropping it where you want it to be positioned. Alternatively you can select an asset or instance from the Asset Browser and then press and hold to preview the resource "in-situ", and if you additionally click the left mouse button you can add the instance and even "paint" it into the room layer by maintaining the button pressed and dragging the mouse. Note that by default instances and sprites that are added to a room in this way will be snapped to the defined room grid settings, but you can hold down / to have the instance follow the mouse precisely and not snap (releasing / will permit snapping again).

For paths you can create a new path layer and path resource right from within the Room Editor- or create a new path resource and then drag it into the main editor window as you would for an instance - and then edit the path and its connections in the editor window too. For tile sets, you can "paint" them in from the tile set editor.

Note that you can select and move or delete multiple assets from the same layer by holding down / + on the ones you want, and most resources in the room editor will permit you to rotate and scale them (singly or as a group) by clicking and holding along the outside edges or corners and dragging the mouse. You can also double click on any single asset (tiles, instances, sprites) to have the editor pan to it and open up the properties window for the asset selected.

Apart from the in-editor tools, you will also see a new drop down menu has been added to the top of the IDE.

These menus are explained below:

  • Layer View: This will re-open the Layer Editor window should you have closed it at any time.
  • Room Properties: This will re-open the Room Properties window should you have closed it at any time.
  • Layer Properties: This will re-open the Layer Properties Window should you have closed it at any time.
  • Instance Creation Order: This will open the Instance Creation Order Window (which can also be opened by using the button in the Room Properties window):
    This window lists all the instances in the room in the order that they will be created (from top to bottom). Should you require a specific instance to be created before any other, you can simply click and drag it to the position your require.
  • Reset Windows On Current Desktop: This will reset the room editor window layout to its default values for the desktop workspace currently focused.

 

  • Tile Editing: When working with a tile layer, this option will be highlighted in the drop down menu. It has the following sub-menu:
    When editing the tile layer you can select multiple tiles by clicking, holding down and dragging the left mouse button , or you can select and group individual tiles using / + , or remove individual tiles from the selection using + . From this menu you can then clear the selection or invert it as well as cut, copy or paste the selection to another layer (or even the same layer but in another position). The final option will re-open the tile set editor docked window should you have closed it.
  • Convert Image To Tile Map: The Convert Image To Tile Map option is a powerful tool that can be used to import a single image and then extract the tiles used from the image and recreate it as a tile map layer in the room editor, creating the required tile set and sprite as part of process. When you select this tool, you will be asked to supply an uncompressed image file (PNG, GIF or BMP) which will then be loaded. On load you will be presented with the Image Import tool:
    Here you are given options on how to split up the tile when creating the tile set, and you can set the cell width and height as well as any offsets required around the edges. You can also set the approximate width (in pixels) for the final sprite that is created. For example, if the tile set is made up of 64x64 tile cells and you set a width here of 200, the final sprite that is created for use in the tile set will be 192 pixels wide (ie: 3 tile cells). If you leave it at the default value of 0, then GameMaker Studio 2 will attempt to make as "square" a sprite as possible with approximately the same number of horizontal and vertical tile cells.

    After setting the way the image is to be split, clicking the swatch beside the Remove Colour option will open the colour picker and permit you to select a colour that is to be removed from the final sprite. This is usually a background colour that you want to remove and setting this swatch to anything other than 100% transparent (alpha 0) will remove the selected colour on import.

    Finally, you have the option to name the sprite, tile set and tile map layer that will be created for you by this tool. When you finalise the import, a sprite with all the images laid out in a grid will be created, as well as a tile set from this sprite. In the room, a new tile map layer will be created and the image reproduced using the generated tile set. Note that the tool will not duplicate tiles and will instead recognise when a cell has image data that coincides with another cell (this includes rotated tiles).
