アニメーションタイル

タイルは基本的にルーム内では動かないものとして扱われますが、GameMaker Studio 2 ではスプライトと同じようにアニメーションさせることも可能です。いやいや、スプライトを使えばいいのでは？もっともな疑問ですが、スプライトには境界ボックスが割り当てられており、そのレンダリング方法から一定の負荷がかかります。しかしタイルはスプライトよりもはるかに処理する負荷が少ないため、高速にレンダリングできるのです。とはいえ良いことづくめではありません。タイルセットでは 2 、4 、8 、16 といったようにフレームの長さが 2 の乗数のアニメーションしか作成できず、タイルのグリッド内にしか置けないことに加えて、タイルセットごとに同じ速さでアニメーションさせなくてはならないのです。このように制限もありますが、アニメーションタイルはとても役に立つ機能であり、動きのない景色や背景を生き生きと彩ることができます。

タイルをアニメーションさせるには、まず必要なタイルを 1 つのスプライト画像に収めたタイルセットを用意しなくてはなりません。スプライトにはフレームを追加せず、単一のフレームにすべてのアニメーション画像を含める必要があります。あとはタイルセットエディタからそのスプライトを指定し、タイルアニメーションボタンをクリックしましょう。このようにアニメーションエディタが開きます。

この例では 1 つの画像に各アニメーションフレームが並べられていますが、アニメーションしないタイルを追加することも可能で、広いタイルセットから一部分だけを選んでアニメーションさせることも可能です。たとえば木や地面、水のタイルがある地形用タイルセットを作るとしたら、水の波紋を再生するタイルの集まりを用意できるでしょう。

タイルをアニメーションさせるには、アニメーション追加ボタンをクリックし、アニメーションライブラリにアニメーションを追加しなくてはなりません。クリックすると空のアニメーションがライブラリに追加され、フレーム数を指定するよう求められます。

フレーム数が指定されると空のフレームが一覧表示され、このように初期フレームが強調されます。

あとはエディタの左側、タイルセットに並んでいるタイルをクリックしてアニメーションに追加するだけです。タイルを追加すると次のフレームに進みますので、そのままクリックし続けてアニメーションを完成させましょう。フレームの横にある「再生」ボタンをクリックするとプレビュー再生もできます。

完成したアニメーションはアニメーションライブラリに組み込まれ、ルームエディタを開くとルームに置けるようになります。なお、アニメーションに含まれているタイルをルームに置くと、そのタイルは追加されたフレームからアニメーションを開始するため、フレーム別に配置することでアニメーションを「ずらして」再生できます。ただし、アニメーション速度は必ず同じになります。これは速度がライブラリのアニメーションごとにではなく、タイルセット全体に設定されるためです。

アニメーションに使われているタイルについては以下の注意点があります。

  • タイルに専用のアニメーションが指定されている（最初のタイルとして設定されているアニメーションがある）場合、そのアニメーションを再生します。
  • タイルに専用のアニメーションがなく、他のアニメーションで使われている場合、そのタイルが最初に使われるフレームからアニメーションを開始します。
  • タイルに専用のアニメーションがなく、複数のアニメーションで使われている場合、アニメーションは再生されません。この場合、最初のタイルとして設定したアニメーションが必要となります。
