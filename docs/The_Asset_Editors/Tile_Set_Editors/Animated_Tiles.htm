アニメーションタイル

タイルは基本的に、ゲーム内のルームでは静的なセルとして見なされますが、GameMaker Studio 2 ではスプライトと同じようにアニメーションさせることも可能です。単にスプライトを使えばいいのでは？もっともな疑問ですが、スプライトには境界ボックスが割り当てられていたり、レンダリング方法の違いがあるため処理上の負荷がかかります。しかしタイルはずっと負荷が少ないため、高速にレンダリングできるのです。とはいえ制限も伴います。タイルセットでは 2 、4 、8 、16 といったようにフレームの長さが 2 の乗数のアニメーションしか作成できず、タイルのグリッド内にしか置けないことに加えて、タイルセットごとに同じ速さでアニメーションさせなくてはなりません。このように制限もありますが、アニメーションタイルはとても役に立つ機能であり、動きのない景色や背景を生き生きと彩ることができます。

アニメーションタイルを作るには、まず必要なタイルがすべて単一のスプライト画像内（フレームは追加せず、単一のフレーム内にすべてのアニメーション画像を含めたもの）に揃っているタイルセットを用意しましょう。それからタイルセットエディタのタイルアニメーションボタンをクリックすると、このようにアニメーションエディタが開きます。

この例はすべてのアニメーションフレームが含まれている 1 枚の画像ですが、画像にはアニメーションしないタイルを含めることができ、1 つの大きなタイルセットから一部分だけを選んでアニメーションさせることも可能です。たとえば木や土地、水のタイルがある広い地形タイルセットなら、水の波紋を再生する複数のタイルを含めることができます。

タイルアニメーションを作るには、まずアニメーション追加ボタンをクリックし、アニメーションライブラリにアニメーションを追加しなくてはなりません。クリックすると空のアニメーションがライブラリに追加され、フレーム数を指定するよう求められます。

すると空のフレームが一覧表示され、初期フレームがこのように強調されます。

そのままエディタの左側にあるタイルセットの画像に戻り、タイルをクリックするとアニメーションに追加でき、次のフレームに進みます。タイルを続けてクリックすればアニメーションを埋められ、フレームの横に表示されている「再生」ボタンをクリックするとプレビューできます。

完成したアニメーションはアニメーションライブラリに組み込まれ、ルームエディタを開くとルームに置けるようになります。なお、アニメーションに含まれているタイルをルームに置くと、そのタイルは追加されたフレームからアニメーションを開始するため、フレーム別に配置することでルームでのアニメーションを「非同期」にできます。1 つだけ気を付けておきたいのは、アニメーションの速度はすべて同じだということです。これは速度をライブラリのアニメーション別にではなく、タイルセット全体に設定するためです。

アニメーションに使われているタイルについては以下の注意点があります。

  • タイルにアニメーションが指定されている場合（最初のタイルとして設定されているアニメーションがある場合）、指定されたアニメーションを使用します。
  • そのタイルだけのアニメーションがなくても他のアニメーションで使われている場合、そのタイルが最初にアニメーション内に現れる位置からアニメーションを開始します。
  • そのタイルだけのアニメーションがなく、他の複数のアニメーションで使われている場合はアニメーションしません。最初のタイルとしたアニメーションを別に用意する必要があるでしょう。
