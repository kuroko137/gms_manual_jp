タイルセットのブラシビルダー

ルームエディタのタイルマップレイヤーにタイルを描こうとすると、最初は 1 つのタイルを選んで描くことになります。しかし大半のタイルセットは部位ごとに分かれており、それらをさまざまな形で組み合わせて地形や物を作るようになっています。たとえば RPG のタイルセットなら、つなぎ合わせることで木や建物といった、大小さまざまな景観を作ることができる地形タイルがあるはずです（ただし扱うタイルの数にもよります）。こうした見た目が異なるタイルをルームレイヤーに置くには何度もタイルを切り替える必要があり、作業をスムーズに進めることが難しくなります。そのような事にならないよう、GameMaker Studio 2 ではタイルブラシを作ることができます。まずはタイルセットエディタ内にあるブラシビルダーボタンをクリックしましょう。

ボタンをクリックするとブラシビルダーが開き、左側には元のタイルセットが、右側には「空」のキャンバスが表示されます。左側からタイルを選択し、右側に描くことで、ルームエディタで使える自分だけのブラシを作ることができます。なお、タイルセットをドラッグすると複数のタイルを選択してキャンバスに描くことができ、 / を押しながらクリックで個別に選択、を押しながらクリックで選択解除できます。

こちらは先ほど取り上げた地形の画像で、単一のタイルセットから作られた 3 つのカスタムブラシが例として示されています。

右側にあるのは、例として作成した 3 つの景観です（画像ではオレンジ色の線で強調されています）。それぞれの景観にすき間を空けていることに注目してください。これはルームエディタ上では、タイルの連続部分が 1 つのブラシとして扱われるためであり、画像ではタイル 1 つ分のすき間を空けて別のブラシにしています。ブラシは左クリックでタイルを描くことで作り、右クリックでタイルを消せます。また、タイルシートおよびブラシキャンバスにカーソルを合わせている間は / を押しながらマウスホイールでズームが、を押しながらドラッグするか中央マウスボタンでドラッグすればスクロールができます。エディタのウィンドウ内にはツールボックスもあり、同じようにズーム倍率を変更できるほか、グリッドの表示と色を切り替えることができます。

右上には選択されているツールが表示され、ブラシのサイズを変更できます。デフォルトのサイズはタイル 1 つ分となる 1 ですが、サイズを上げると以下の画像のように、選択中のタイルを並べたより大きなブラシで描けます。

ツールボックスからはタイルセットエディタのさまざまな作業に使えるツールを選択できますが、一部のツールはオートタイルライブラリにブラシが存在しないと利用できません。以下は各ツールの概要となります（なお、ルームエディタでタイルレイヤーを選択すると、ルームのワークスペース上部に同じツールボックスが描画用のツールとともに表示されます）。

鉛筆 鉛筆ツールを使うと、左クリックで選択中のタイルをブラシウィンドウに描くことができます（右クリックでタイルを消せます）。
消しゴム 消しゴムツールを使うと、ブラシウィンドウに置かれたタイルを左クリックで消すことができます。実際にはタイルのインデックス値を、「空のタイル」のインデックス値として予約されている 0 に置き換えています。
選択 選択ツールを使うと、ブラシウィンドウの編集範囲を決めることができます。左クリックをしてドラッグすると長方形の範囲を選択でき、さらに / を押しながらドラッグで選択範囲を追加、を押しながらドラッグで削除できます。すべて解除したくなったらを押してください。ブラシウィンドウに選択範囲がある場合、ほかのツール（鉛筆、反転、回転など）はその範囲内でのみ扱うことができます。また、ウィンドウで選択したタイルはコピー（ / + "C" ）、切り取り（ / + "X" ）、貼り付け（ / + " V" ）も可能で、この場合は鉛筆ツールに切り替わり、選択したタイルで描き始めることができます。
オートタイル クリックするとオートタイルが有効となります。有効にするとオートタイルライブラリのタイルが選択可能となり、ブラシウィンドウにタイルを描いたとき、外周部分のタイルを自動で補正してくれるようになります（ただしオートタイルタブで正しく設定されている場合に限ります）。なお、タイルセットからタイルを選んでもオートタイルライブラリに加えることはできず、標準の鉛筆ツールに戻されます。
上下反転 上下反転ツールを左クリック（またはキーボードショートカットの "X" を入力）すると描画ツールを変えることなく、タイルセットから選択したタイルを縦に反転できます（カスタムブラシを使っている場合はブラシ全体を反転します）。ブラシウィンドウ内のタイルを選択している場合、その範囲内のタイルをすべて上下反転します。
左右反転 左右反転ツールを左クリック（またはキーボードショートカットの "Y" を入力）すると描画ツールを変えることなく、タイルセットから選択したタイルを横に反転できます（カスタムブラシを使っている場合はブラシ全体を反転します）。ブラシウィンドウ内のタイルを選択している場合、その範囲内のタイルをすべて上下反転します。
回転 回転ツールを左クリック（またはキーボードショートカットの "Z" を入力）すると描画ツールを変えることなく、タイルセットから選択したタイルを時計回りに 90° 回転できます（カスタムブラシを使っている場合はブラシ全体を回転します）。ブラシウィンドウ内のタイルを選択している場合、その範囲内のタイルをすべて回転します。

ツールの下側には、現在のタイルセット画像から作られたオートタイルおよびアニメーションタイルを選択できる領域があります。タイルセットに使用されている 1 枚のスプライトにはとても多くのセル画像を含めることができますが、それらを組み合わせることで上記の領域に表示されるカスタムブラシを作成でき、動かない普通のタイルと組み合わせたカスタムブラシを作ることも可能です（アニメーションタイルに関しては、ライブラリと元のタイルセットのどちらから選択しても必ずアニメーションするためご注意ください）。

必要なブラシをすべて設定できたら、ルームエディタのタイルマップレイヤーに作成したブラシで描けるようになります。
