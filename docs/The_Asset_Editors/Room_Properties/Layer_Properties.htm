レイヤーの種類とプロパティ

ルームエディタで作成したものはすべてレイヤーの中に置かれます。レイヤーはいずれもレイヤーエディタから追加・削除したり、整理することができます。レイヤーにはタイルレイヤー、パスレイヤー、インスタンスレイヤーといった種類があり、それぞれに専用の設定、仕組みがあります。数には上限がなく、どんな種類のレイヤーでも複数追加可能です。レイヤーには種類ごとにプロパティウィンドウがあり、レイヤーの種類によって設定できる内容は異なります。レイヤーは表示の切り替えも可能で、非表示にしたレイヤーは実行時に処理されません（ただし専用のレイヤー関数によっていつでも再表示できます）。

レイヤーエディタはこのような見た目をしています。上部はレイヤーリスト、下部はレイヤー作成ボタンとなります。表示されている名前をゆっくりダブルクリックするとレイヤーの名前を変更でき（英数字および "_" 記号のみ受け付けます）、目アイコンをクリックするとレイヤーの表示を切り替えられます。親ルームから継承されたレイヤーがある場合、下のボタンから継承の切り替えができますが、すべてのレイヤーに影響するためご注意ください（こちらで詳しく解説されています）。なお、または / を押しながらレイヤーをクリックすると複数選択でき、まとめて複製したり削除できるようになります（ボタンだけでなく右クリックメニューからも同じ操作が可能です）。

レイヤーの並びを整理したくなったら、フォルダボタンをクリックしてレイヤーフォルダを作成し、レイヤーをフォルダの中に入れましょう。削除ボタンを押した場合は、選択中のレイヤーをまとめて削除できます。すべてのレイヤーには深度値という、ルームにレイヤーの内容がレンダリングされたときに、そのレイヤーが描画順のどこに置かれるかを決める値があります。レイヤーは高い深度から低い深度へと描画されるため、深度が-100 のレイヤーがある場合、深度が 200 のレイヤーより前に描画されます。

重要！レイヤーの深度には-16000 から 16000 までの上限値があり、上限外のレイヤーに置かれたインスタンスは描画されません。ただしレイヤー上にあるもののイベントはすべて通常どおりに実行されます。

レイヤーを右クリックすると、以下のレイヤーメニューが開きます。

このメニューからはレイヤープロパティウィンドウを開いたり、レイヤーの削除、名前変更、さらにはサブレイヤーを追加できます。サブレイヤーを追加した場合、現在のレイヤーにはサブレイヤーが付属していることを示す「フォルダ」アイコンが与えられ、新しいレイヤーは現在のレイヤーの下、右にタブを空けて作成されます。あとはサブレイヤーに親レイヤーのプロパティを継承させるか選択したり、親レイヤーの表示を継承させるかを設定することができます。

なお、ウィンドウ内のレイヤーは上下にドラッグすると順番を変更できるほか、を押しながらクリックするとあるレイヤーから別のレイヤーまで複数選択、移動できたり（間にあるすべてのレイヤーごと）、 / を押しながらクリックで個別選択もできます。レイヤーをレイヤーフォルダの上にドラッグするとフォルダの中に移動し、そのフォルダのサブレイヤーとなります。また、ロックボタンをクリックするとレイヤーをロックし、間違って編集できないようになります。

メモ: エディタのウィンドウではそれぞれ異なるアセットをレイヤーに配置することになりますが、"P" を押している間はアセットをすぐに選択でき、そのアセットが配置されているレイヤーに切り替えることができます。

レイヤーの種類、およびレイヤーの編集可能プロパティについては以下でそれぞれ説明されています。

 

背景背景

このボタンをクリックすると背景レイヤーが追加され、背景のプロパティウィンドウが開きます。

背景レイヤーを作成すると最初は空となっており、背景色が描画されます。これは背景レイヤーの画像となるスプライトが割り当てられていないためです。なお、色をクリックするとカラーピッカーを開いて背景色を変更できます。スプライトが指定されている場合、水平タイル、垂直タイル、画像をルーム全体にストレッチさせるかの切り替えが可能となります。

背景を決められた方向にスクロールさせたいなら、水平・垂直速度を設定しましょう（ゲームステップごとに一定ピクセル動くようになります）。また、X / Y のオフセット値を設定すれば背景画像の位置を変えることができます。指定したスプライトリソースに複数のサブ画像がある場合、背景のアニメーション速度を設定できますが、デフォルトではスプライトエディタの設定値にロックされます。レイヤーの深度も指定できますが、同じくデフォルトではロックされています。値を変更するだけなら、レイヤーエディタからレイヤーをドラッグして順番を変えるだけで可能です。

ウィンドウの一番上のボタンをクリックした場合、背景の継承が切り替わります。

 

インスタンスインスタンス

このボタンをクリックするとインスタンスレイヤーが追加され、インスタンスのプロパティウィンドウが開きます。

このウィンドウには、レイヤーに追加されているインスタンスのリストが表示されます。アセットブラウザを開いてオブジェクトをルームまでドラッグすると、レイヤーに新しいインスタンスを追加できます。またはアセットブラウザからインスタンスを選択後、を押している間はリソースがその場にプレビュー表示され、そのまま左クリックするとインスタンスを追加、ドラッグすると「描く」ように連続して追加できます。

リストに並んでいるインスタンスはドラッグして自由に並び替えできますが、あくまでルームエディタのリスト表示を整理できるだけであり、インスタンスが生成される順番を変更したい場合は、ルームプロパティのウィンドウ、またはルームの IDE メニューから生成順序ウィンドウを開かなくてはなりません。

ルームエディタまたはプロパティに表示されているインスタンスをダブルクリックすると、新しいウィンドウを開いて個々のインスタンスの性質を変更できます。

インスタンスには "inst_38B0BFF" のようなデフォルト名が与えられますが、この名前は自由に変更できます（英数字および "_" 記号のみ使用でき、スペースは受け付けません）。注意点として、ここで指定した名前は定数として扱われるため（自分でつけた名前か IDE によるデフォルト名かを問わず）、インスタンスごとに固有の名前にしなくてはなりません。この名前はコードからインスタンスを識別するための名前となりますが、識別できるのはインスタンスが同じルームに存在する場合のみです。

インスタンス名の下にある各ボタンからは、インスタンスを親ルームから継承させるかの設定、オブジェクトの変更、オブジェクトエディタを開いて元のオブジェクトを呼び出せるほか、オブジェクト変数の編集と確認ができます。インスタンスが最初からもつ特徴を変えることも可能で、ブレンドされる色（デフォルトでは白色）、回転および反転、縦横のスケール変更、ルーム内の位置、スプライトの画像速度、初期フレームといったものを設定できます。

他に設定できるものは、インスタンスの生成コードです。このコードはオブジェクトの Create イベントの直後に実行され、そのインスタンスだけのものとなります（たとえば元のオブジェクトで設定した変数を、インスタンスの生成コードから独自の値に変更するといったことが可能です）。ボタンをクリックするとスクリプトウィンドウがインスタンスの横に開き、関数を追加したり、変数を変更できるようになります。コードが実行されるタイミングはインスタンスが生成され、Create イベントが実行された直後です。なおインスタンスの継承とは別に、生成コードの継承を切り替えることも可能です。

レイヤー上のインスタンスは、ゲーム作成時に自動でエクスポートするようフラグ付けされます。とはいえ継承を利用していたり、常に存在し続けるデバッグ用オブジェクトが用意されている場合は、完成したゲームのルームに入れたくないインスタンスが見つかるかもしれません。このような場合、リストのインスタンスの横にある "エクスポート" のチェックを外せば、実行ファイルの一部として出力されなくなります。ただし大きな注意点があり、チェックを外したインスタンスをコードから参照している場合、ゲームが正常に実行されなくなる恐れがあります。そのため注意して扱ってください。

 

タイルマップタイルマップ

このボタンをクリックするとタイルマップレイヤーが追加され、タイルマップレイヤーのプロパティウィンドウとタイル編集ウィンドウが開きます。

タイルマップレイヤーのプロパティでは、レイヤーの継承切り替え、使用するタイルセットの変更ができ、レイヤーのグリッドをルームの (0, 0) に揃えたくない場合は X / Y のオフセットを設定できます。レイヤーがレンダリングされる深度を設定することもでき（専用の継承ボタンもあります）、ロックをすれば不要な編集を防ぐことが可能です。

レイヤープロパティのウィンドウを開くとさらにタイル編集ウィンドウも開き、現在のタイルセットが表示されます（ウィンドウを間違って閉じてしまった場合、IDE 上部のメニューからルーム > タイル編集というように選択すれば開き直せます）。クリックで選択したタイルはブラシになり、ルームエディタのキャンバス上をクリックすれば現在のレイヤーに描くことができます。複数のタイルで描きたくなったら、タイルセットのタイルをドラッグするとまとめて選択できます。ブラシに追加したいタイルが離れている場合、 / を押しながら左クリックでタイルを個別に選択し、カスタムブラシにできます（を押しながらクリックでタイルの除外もできます）。こうして作られたカスタムブラシは一時的にしか使えませんが、ブラシタブで作ったものはタイルマップレイヤーに保存されるため無くなりません。 

タイル編集ウィンドウではマウスホイール、上部の各ボタンによってタイルのズーム倍率を変更できるほか、中央ボタンによるドラッグまたはを押しながらドラッグでスクロールできます。このウィンドウにはグリッドオプションもあり、有効にしてタイルを区切ったり、グリッドそのものの色を変更できます。なお、グリッドを非表示にしてもタイルは必ずスナップします。

このウィンドウにはライブラリタブもあり、ブラシ、アニメーションタイル、オートタイルを選択できます。これらに関してはタイルセットエディタのページで取り上げられています。

タイルレイヤーの選択時は、ルームエディタの上部にタイル用のツールが表示されます。以下はその一覧です。

鉛筆 鉛筆ツールを使うと、選択中のタイルをブラシとしてルームエディタに左クリックでタイルを描くことができ、右クリックするとタイルを消せます。ルームエディタのステータスバーにはタイルのインデックス値が表示され、複数のタイルをブラシとして選択している場合、すべてのタイルのインデックスが一覧となって表示されます。
消しゴム 消しゴムツールを使うと、ルームエディタのタイルを左クリックで消すことができます。実際にはタイルのインデックス値を、空のタイルのインデックス値として予約されている 0 に変更しています。
塗つぶし 塗りつぶしツールではタイルマップレイヤー全体、あるいは選択中のタイルを現在のタイルブラシで塗りつぶすことができます。ブラシには単一のタイル、複数のタイルからなる複合ブラシ、さらにはアニメーションライブラリのタイルを利用でき、レイヤー上をクリックすると指定したブラシで塗りつぶされます。選択範囲がない場合は空のタイル、または同じインデックスのタイルがつながっている部分を塗りつぶします。選択範囲がある場合、範囲内のすべてを塗りつぶします。
直線 直線ツールを選択すると、ドラッグしてタイルの直線を描けるようになります。タイルマップレイヤーに選択範囲がある場合、範囲内にのみ直線で描けます。このツールでは 1 つのタイルのブラシ、複数のタイルのブラシ、ライブラリのアニメーションタイルのいずれも利用できます。
長方形 長方形ツールを選択すると、ドラッグしてタイルで塗りつぶした長方形を描けるようになります。タイルマップレイヤーに選択範囲がある場合、範囲内にのみ長方形で描けます。このツールでは 1 つのタイルのブラシ、複数のタイルのブラシ、ライブラリのアニメーションタイルのいずれも利用できます。
選択 選択ツールを使うと、タイルレイヤーの編集範囲を決めることができます。左クリックをしてドラッグすると長方形の選択範囲ができ、 / を押しながらドラッグで選択範囲の追加、を押しながらドラッグで選択範囲の削除ができます。選択範囲をすべて消したいときはを押しましょう。タイルレイヤーに選択範囲がある場合、ほかのツール（鉛筆、反転、回転など）はその選択範囲内でのみ扱うことができます。また、ルームエディタで選択されているタイルはコピー（ / + "C" ）、切り取り（ / + "X" ）、貼り付け（ / + " V" ）も可能で、この場合は鉛筆ツールに切り替わり、選択中のタイルでルームのレイヤーに描くことができます。
オートタイル このツールをクリックすると、オートタイル形式で描けるようになります。オートタイルライブラリのタブから選んだタイルでルームレイヤーに描くと、外側部分のタイルを自動で調整してくれるようになります（ただしオートタイルタブで正しく設定されている場合に限ります）。なお、タイルセットからタイルを選んでもオートタイルライブラリに加えることはできず、標準の鉛筆ツールに戻されます。
継承関係 レイヤーに置かれたタイルの継承関係を切り替えることができるツールです。このツールを使うには、まず開いているルームが子ルームであり、レイヤープロパティからレイヤーの継承をオンに切り替えなくてはなりません。この状態だと、ルームエディタでは継承されたタイルがすべて赤く表示され、右クリックしてタイルの継承を無効に、左クリックして有効にできるようになります。こうすることで元のルームにあるものをそのまま流用でき、一部のタイルだけを変更することができます。
上下反転 上下反転ツールを左クリック（またはキーボードショートカットの "X" を利用）すると、描画ツールを変更することなくタイルセットから選択したタイルを縦に反転できます（カスタムブラシを使っている場合はブラシ全体を反転します）。レイヤー上のタイルを選択している場合、その範囲内のタイルをすべて反転します。
左右反転 左右反転ツールを左クリック（またはキーボードショートカットの "Y" を利用）すると、描画ツールを変更することなくタイルセットから選択したタイルを「横」に反転できます（カスタムブラシを使っている場合はブラシ全体を反転します）。レイヤー上のタイルを選択している場合、その範囲内のタイルすべてを左右反転します。
回転 回転ツールを左クリック（またはキーボードショートカットの "Z" を利用）すると、描画ツールを変更することなくタイルセットから選択したタイルを時計回りに 90° 回転できます（カスタムブラシを使っている場合はブラシ全体を回転します）。レイヤー上のタイルを選択している場合、その範囲内のタイルをすべて回転します。

なお、選択ツールでルームエディタのキャンバス上にあるタイルを複数選択し、切り取りまたはコピーすると、一時的なカスタムブラシとして扱うことができます。

 

パスパス

このボタンをクリックするとパスレイヤーが追加され、パスのプロパティウィンドウが開きます。

このウィンドウでは、アセットブラウザから選択したパスをレイヤーに割り当てることができ、さらに接続タイプや（スムース補間または直線）、パスを閉じるかどうか、線をつなぐ精度（滑らかさ）を設定することができます。ほかには継承の切り替えや、エディタ内で見やすくなるようパスの色を指定することができます（デフォルトでは赤色）。パスが割り当てられていない場合でも、新規作成ボタンをクリックすれば新しいパスを作成できます。パスレイヤーには深度値もありますが（デフォルトではロック中）、パスを利用しても実際のゲームでは何もレンダリングされません。

また、左クリックでパスにポイントを追加したり、右クリックでポイントを削除することができます。ポイントはクリックして追加でき、ドラッグして引っぱることができます。いずれの操作もルームエディタから可能で、ルームを見ながら簡単にパスを作ることができます（もちろんパスエディタからでも同じパスを編集できます）。 / を押しながら別々のポイントをクリックすると、それらを選択してまとめて動かしたり、メニューから削除することができます（以下で説明）。

パスの操作中にルームエディタのウィンドウを右クリックすると、以下のメニューが開きます。

メニューからはパスの方向を逆にしたり（見た目は特に変わりませんが、ポイントの順番が逆になるため最初のポイントが最後のポイントになります）、縦か横に反転することができます。選択中のポイントを削除することも可能です。なお、"T" 、"S" 、"R" を押しながらドラッグするとパス全体を移動、拡大縮小、回転できます。

 

アセットアセット

このボタンをクリックするとアセットレイヤーが追加され、アセットレイヤーのプロパティウィンドウが開きます。

アセットレイヤーは 1 つの画像アセット（スプライトおよびシーケンス）であり、インスタンスとは異なるものとしてルームに配置されます。このレイヤーにはアセットブラウザからドラッグして画像アセットを追加でき、を押しながらクリックすると描くように追加できます。スプライトアセットはタイルと似ていますが、サブ画像があれば完全なアニメーションを再生でき、タイルのような制限もないためグリッドに押し込まれることもなく、アニメーションの長さと速度も自由に変更できます。シーケンスはスプライト、サウンド、インスタンスを組み合わせたアセットで、ルームに入ったタイミングで再生されます。

アセットレイヤーにアセットを置くとグリッドのスナップ値（ルームエディタの上部から設定可能）に合わせてスナップしますが、自由に配置したいときはグリッドを無効にするか、 / キーを押しながら左クリックしましょう。これらのキーを押している間はアセットを好きな位置に置けるようになり、グリッドの分かれ目にスナップしなくなります。

アセットをルームに配置したら、ダブルクリックしてアセットのプロパティウィンドウを開いて、アセットの見た目をある程度コントロールできます。

プロパティではアセットに固有の名前をつけたり、親ルームからの継承設定、表示するスプライトおよびシーケンスの変更ができます。シーケンスにおいては、アニメーション速度と再生ヘッドの初期フレームを設定できるほか、ブレンドする色（デフォルトでは白色）、ルーム内での位置を設定できます。スプライトアセットの特徴も変更でき、ブレンドする色（デフォルトでは白色）、回転、反転といった設定ができます。スプライトは縦横のスケールを変更したり、ルーム内での位置も設定可能です。スプライトにサブ画像がある場合は、アニメーション速度、表示される初期フレームを指定できます。

レイヤー上のアセットは、ゲーム作成時に自動でエクスポートするようフラグ付けされます。とはいえ継承を利用している場合などでは、完成したゲームのルームに入れたくないスプライトが見つかるかもしれません。このような場合、レイヤープロパティのリストに表示されているスプライトの左にある "エクスポート" のチェックを外せば、実行ファイルの一部として出力されなくなります。ただし大きな注意点があり、チェックを外したアセットをコードから参照している場合、ゲームが正常に実行されなくなる恐れがあります。そのため注意して扱ってください。
