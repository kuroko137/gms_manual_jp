イメージエディタ

イメージエディタでは専用のスプライトを作成したり、インポートまたは作成済みのイメージ（画像）を編集できます。画像編集ソフトを使ったことのある方にとっては見慣れたツールですが、GameMaker Studio 2 のイメージエディタには、なかなか気付きにくい機能がいくつかあります。

  • 複数のフレームを用意すれば、アニメーションさせながら描くことができます！フレーム操作バーの再生ボタンをクリックし、ブラシを選択して描いてみてください。エディタ内でフレームのアニメーションが再生され、表示中のフレームに描かれた内容はそのままイメージに反映されます。ちょうどいいフレーム速度になるまで試す必要はありますが、コツを掴んだらあっという間に動きのあるスプライトやエフェクトを作れるでしょう。
  • イメージエディタのタブを引っぱると、ワークスペースのメインウィンドウから独立したウィンドウに切り離すことができます。複数のモニターを使用しているなどの場合に便利です。
  • 選択範囲をコピーすると、コピーした内容がそのままブラシリストに追加されます。つまり画像の一部分だけを選び、直線や多角形、絵筆ツールのブラシとして使えるということです。このブラシは大きさを変えたり、色を重ねることも可能です。なお、イメージエディタの貼り付け機能はエディタ内に限定された機能ではありません。そもそもコピーした内容はブラシリストに追加されるため、そのブラシを選択すれば普通に貼り付けができるはずです。イメージエディタの貼り付け機能は、外部のプログラムからキャプチャーした画像をエディタに貼り付けするためのものです。
  • スプライトにはレイヤーエディタから複数のレイヤーを追加できます。レイヤーは順番を変えたりフォルダに入れられるため、複雑な画像も作りやすくなるでしょう。
  • キーボードの "1" および "2" キーを押すと、エディタのサブイメージ（フレーム）をすぐに切り替えることができます。

これらはイメージエディタに用意されている高度な機能のほんの一部です。機能のすべてを知りたいなら、下の項目を開いてエディタの各領域についての説明をご覧になるといいでしょう。

フレームビューフレームビュー

この領域にはアニメーションする順番でフレームが表示されます。左のボタンをクリックすると新しいフレームを追加でき、イメージメニューを開いても同じように追加できます。フレームをドラッグした場合は、タイムライン上の位置を変更することができます。フレームビューの左上からはオニオンスキンの切り替えが可能です。有効にすると現在のフレームの裏に、前後のフレームを半透明で重ねて表示させることができます。この機能の良いところは、遠くのフレームほど薄く表示されるためアニメーションの進み方を確認しながら描けるということです。

オニオンスキンを有効にするとフレームの追加操作が表示され、ドラッグして前後のフレーム表示数を設定できるようになります。

ページの下側で説明されているように、ビューのコンテキストメニューからはオニオンスキンの設定値を変更できます。

オニオンスキンボタンの下にあるのはループ制御ボタンで、スプライトのループ再生（最後のフレームになったら 1 フレーム目に戻る）、ピンポン再生（最後のフレームになったら逆方向に進む）を切り替えることができます。すぐ下にある再生ボタンを押した場合は、アニメーションのプレビューを開始 / 停止できます。

フレームをクリックすると編集するフレームを選択でき、フレームの角に表示される削除ボタンをクリックするとスプライトからそのフレームを削除できます。また、 / を押しながらクリックすれば複数選択となり、そのまま削除ボタンを押した場合はすべての選択フレームを削除できます。また、フレームは / + "X" で切り取り、 / + "C" でコピー、 / + "V" で貼り付けといったように標準キーボードショートカットでも編集できます。

 

フレーム情報フレーム情報

フレームの情報が表示されるバーで、左の入力欄からはフレームのアニメーション再生速度を設定できます。フレーム速度は「フレーム / 秒」または「フレーム / ゲームフレーム」のいずれかを基準として設定します。右に表示される数値は合計フレーム数、選択中のフレーム番号です。なお、このバーは左のアイコンをクリックすれば閉じる（非表示にする）ことができます。イメージエディタ設定ではこのバーを最初から開くか、閉じるかどうかを設定できます。

注意点として、バーに入力した速度はゲーム内におけるスプライトのアニメーションにも適用されます。レイヤー上のスプライトアセット、インスタンスに割り当てたスプライトのどちらにも適用されますが、対応する関数ないし変数によって実行時のアニメーション速度を変化させることも可能です。

 

キャンバス操作部キャンバス操作部

ここでは描画キャンバスの表示をさまざまな形で変更できます。

  • キャンバスのグリッド切替: キャンバスのグリッドを有効 / 無効にします。これはキャンバスを区切るために GameMaker Studio 2 側で表示されるグリッドで、デフォルトのサイズは 1px となります。グリッドメニューアイコンをクリックした場合、次のようなグリッド設定が開きます。
    ここではグリッドの色、透明度、さらにグリッドのセル値を X/Y 軸ごとに設定できます。また、グリッドのスナップを切り替えることも可能です（デフォルトでは無効）。
  • キャンバスのズーム操作: 各ボタンからキャンバスのズーム倍率を変更できます。拡大縮小のほかに、ボタンをクリックすればキャンバスを等倍表示にリセットできます。ウィンドウにフィットボタンをクリックした場合、キャンバス全体をエディタのワークスペース内に収めることができます（同時に拡大縮小されます）。なお、 / を押しながらマウスホイールでも拡大縮小、 / + でもキャンバスを等倍表示にリセットできます。
  • キャンバスの分割: クリックするとキャンバスの表示が水平分割、垂直分割、単独表示の順に切り替わります。分割するとキャンバスごとに操作パネルが表示されるため、片方をズームしてもう片方をズームアウトするといったことも可能です。なお、キャンバスで行った編集はもう片方のキャンバスにも反映されます。キャンバスを別の表示に切り替えたいなら、分割設定ボタンをクリックして直接指定するか、表示したいモードになるまで分割ボタンを繰り返しクリックしましょう。

 

描画キャンバス描画キャンバス

イメージの編集を行うメインウィンドウです。右側に並んでいるツールを使って選択中のレイヤーに絵を描いていきます。左クリックまたは右クリックで描くことができ（カラーピッカーから左右の色を指定できます）、エディタの右上部分でブラシを変えることができます。また、 / を押しながらマウスホイールでズーム操作、 / + でキャンバスを等倍表示にリセットでき（上記のキャンバス操作部からも実行できます）、中央ボタンでドラッグ、あるいはを押しながらドラッグで画面移動ができます。

 

キャンバスの数値キャンバスの数値

下側に表示されているバーからは、マウスの位置、キャンバスのサイズ、選択範囲の幅といった数値を確認できます。

 

ブラシブラシ

ここでは描画用のブラシを選択できます。デフォルトでは四角ブラシ、丸ブラシを各サイズから選べますが、自分用のブラシを作ることもできます。レイヤーに描画し、ブラシとして使いたい部分を選択して / + "C" でコピーしてください。コピーした部分がそのまま新しいブラシとして追加されます。直線などのツールも利用可能で、他のブラシと同じように扱うことができます。

なお、新しいブラシの色はデフォルトで白となりますが、カラーピッカーから別の色に変更した場合、その色をブレンドして描くことができます。リストからブラシを削除したい場合は、ブラシをクリックしてキーを押してください。

 

カラーピッカーカラーピッカー

カラーピッカーの上部に並んでいるのは既定の色見本で、左クリックまたは右クリックで選択すると、左右のどちらかにその色を割り当てることができます （2 つの色の横にある矢印アイコンをクリックすると色の入れ替えも可能です）。選択した色はマウス画像の隣に表示され、色をクリックするとカラーピッカーウィンドウを開くことができます。

このウィンドウではスライダーなどを操作したり、ARGB 値または HSV 値を入力して色を決めることができます。なお、エディタの色見本をダブルクリックすれば、カラーピッカーツールを開いてスプライトの既存パレットを編集できます。この場合は、「OK」をクリックしてカラーピッカーを閉じるとパレットに新しい色が設定され、編集中のスプライトアセットと一緒に保存されます。編集したカラーパレットは、カラーパレットの設定メニューから別のスプライトに適用することも可能です。

設定メニューを利用すれば、カスタマイズしたカラーパレットをアセットブラウザの他のスプライトに適用したり、他のスプライトのカラーパレットをインポートできます。こうしたスプライトのパレット機能については、YoYo Games tech blog でより詳しく紹介されています。

  • GMS2 のあまり知られていない機能

 

ツールツール

以下は GameMaker Studio 2 に最初から用意されているツールの一覧です。ツールは左クリック、右クリックのどちらでも使用でき、その多くにキーボードショートカットがあります（ここには重要なショートカットのみ記載しますが、ショートカットのページからすべてのショートカットを確認できます）。また、ほとんどのツールにはイメージエディタの上部分にこのような追加設定が表示されます。

鉛筆 鉛筆ツールでは選択した色、ブラシでクリックして描くことができます。イメージエディタ上部のバーからサイズを変更したり、スムース（ぼかし補間）の切り替えも可能です。また、と "Z" を押しながらドラッグすると現在のブラシのサイズ比を変えることができます。右上に並んでいる既存のブラシでも可能です。
消しゴム 消しゴムツールを使うと、選択中のブラシでクリックした範囲を消すことができます。このツールに切り替えるとエディタ上部のバーに設定が追加され、ブラシのサイズを変えたり、スムース（リニア補間）の切り替えができるようになります。
ペンキ / 塗りつぶし ペンキ（塗りつぶし）ツールでは画像を塗りつぶすことができます。塗りつぶされる範囲は、エディタの上部バーから設定できる透明度の許容値によって変わります。
色除去 色除去ツールを選択してから画像のピクセルを左か右クリックすると、選択したピクセルと同じ色のピクセルをすべて除去することができます（正確には透明度 0 の黒色に置き換えられます）。上部のバーにあるスライダーからは色の許容範囲を変更でき、低くすればするほどクリックした色と正確にマッチするようになり、高くすればするほどクリックした色と近い色にまで拡がるようになります。選択ツールによって範囲が選択されているなら、その中のピクセルをクリックした場合にのみ有効となり、選択範囲内のピクセルだけが除去されます。
色置換 色置換ツールを選択してから画像のピクセルを左か右クリックすると、同じ色のピクセルをマウスに割り当てた色で置き換えます。上部のバーにあるスライダーからは色の許容範囲を変更でき、低くすればするほどクリックした色と正確にマッチするようになり、高くすればするほどクリックした色と近い色にまで拡がるようになります。選択ツールによって範囲が選択されているなら、その中のピクセルをクリックした場合にのみ有効となり、選択範囲内のピクセルだけが色を変更されます。
直線 直線ツールでは 2 つの地点をもとに線を引くことができます。マウスボタンでクリックした位置が開始点となり、ボタンを離した位置で線が繋がれます。直線はブラシを選択して描くことができ、エディタの上部バーからブラシの設定値（サイズやリニア補間など）を変更できます。
長方形 このツールでは塗りつぶし、または輪郭線のみの長方形を描くことができます。アイコンの左上部分をクリックした場合は輪郭線のみ、右下部分をクリックした場合は塗りつぶしとなります。輪郭線と中身をどちらも描画したい場合は、イメージエディタ上部のバーから「両方」を選択してください（またはを押しながらクリックすれば、どちらの部分も選択できるようになります）。このバーではブラシのサイズ、スムースの設定も可能です。長方形はブラシを選択して描くことができます。なお、を押しながらクリックすると、クリックした位置を中心として描画できます。
楕円形 このツールでは塗りつぶし、または輪郭線のみの楕円形を描くことができます。アイコンの左上部分をクリックした場合は輪郭線のみ、右下部分をクリックした場合は塗りつぶしとなります。輪郭線と中身をどちらも描画したい場合は、イメージエディタ上部のバーから「両方」を選択してください（またはを押しながらクリックすれば、どちらの部分も選択できるようになります）。このバーではブラシのサイズ、スムースの設定も可能です。楕円形はブラシを選択して描くことができます。なお、を押しながらクリックすると、クリックした位置を中心として描画できます。
多角形 このツールでは塗りつぶし、または輪郭線のみの多角形を描くことができます。アイコンの左上部分をクリックした場合は輪郭線のみ、右下部分をクリックした場合は塗りつぶしとなります。輪郭線と中身をどちらも描画したい場合は、イメージエディタ上部のバーから「両方」を選択してください（またはを押しながらクリックすれば、どちらの部分も選択できるようになります）。このバーではブラシのサイズ、スムースの設定も可能です。左か右クリックをすると多角形にポイントを追加できます。描きたい形になったら他のツールに切り替えてみてください。変更が確定され、画像が生成されます。多角形はブラシを選択して描くことができます。を押せば、いつでも多角形の編集を中断することができます。
円弧 円弧ツールでは選択したブラシで円弧（スプライン）を作成できます。左右のどちらかでクリックしてポイントを設定し、ドラッグして円弧を生成、そしてマウスボタンを離してからポイントと曲線の追加を繰り返します。円弧は選択したブラシで描かれ、エディタ上部のバーからブラシのサイズ、スムースの設定ができます。上部のバーからは円弧の滑らかさも設定可能で、数値を下げるほどカクつき、上げるほど滑らかになります。デフォルトの設定値は 10 です。
テキスト
テキストの入力ツールを使うと、新しいレイヤーにテキスト画像を作成できます。イメージエディタ上部のバーからフォントやその書式を設定できます。
スポイト スポイトツールを使うと、編集している画像の色を取得できます。
長方形選択 このツールを使うと長方形の選択範囲を作れます。を押すか「イメージ」メニューを開くと選択を解除できます。
手書き選択 このツールを使うと長方形の選択範囲を作れます。「イメージ」メニューを開くか、キーで選択を解除できます。
自動選択 自動選択ツールでは、エディタ上部のバーから設定できる許容値をもとに同じ色の範囲を選択できます。「色のみ」をチェックするとアルファ値を対象としなくなります。を押すか「イメージ」メニューを開くと選択を解除できます。
回転 選択ツールで範囲を選択してからこのツールをクリックすると、選択範囲を 0° から 360° の角度で回転させることができます。範囲を選択していない場合は、現在のブラシを回転できます。回転させるにはドラッグするか、上のバーから数値を直接入力してください。を押すか、別のツールに切り替えると変更が確定されます。なお、変更を確定しても元の画像またはブラシはそのままで、選択部分またはブラシから設定した回転量の新しいブラシが作成されます。また、ブラシはキーボードショートカットの "Z" でも 90° 単位で回転できます。この場合は新しいブラシが作成されず、現在のブラシの表示にのみ反映されます。
左右反転 左右反転ツールを選ぶとブラシを横に反転できます。新しいブラシが作成されるのではなく、表示中のブラシにのみ反映されます。
上下反転 上下反転ツールを選ぶとブラシを縦に反転できます。新しいブラシが作成されるのではなく、表示中のブラシにのみ反映されます。
移動 移動ツールを選んでからドラッグすると、レイヤー内の画像を動かすことができます。注意点として、キャンバスの外に出されたピクセルは表示されなくなり、ツールが変更されると描画キャンバスのサイズで切り抜かれます。また、選択範囲がある場合はレイヤー全体ではなく、選択範囲内のピクセルを動かせるようになります。

 

レイヤーエディタレイヤーエディタ

イメージエディタではレイヤーを作成し、画像を複数のパーツに分けることができます。レイヤーはそれぞれ個別にブラシで描いたり、ツールで操作することができます。レイヤーはアイコンをクリックするだけで作成でき、アイコンをクリックすると削除できます。レイヤーはドラッグして順番を並び替えることができ、ボタンをクリックしてフォルダを作成すれば、その中にレイヤーをドラッグして整理することもできます。レイヤーをダブルクリック（または右クリックして「レイヤープロパティ編集」を選ぶ）した場合は、このようにレイヤーのプロパティウィンドウが開きます。

このウィンドウではレイヤーの名前のほかにブレンドモード（下層のレイヤーとの合成モード）を設定でき、通常、加算、減算、乗算のいずれかを指定できます。あとはレイヤーの透過値（透明度）をスライダーで設定できますが、実際には各ピクセルのアルファ値がこの数値に加えられます。たとえばピクセルの透過値が 50%、レイヤーの透過値が 50%の場合、ピクセルは結果として 25%の透過値で表示されます。

ImageImage

The Image menu has the following options (some of which may or may not be available depending on the currrent editor focus):

  • Cut Frame / Selection - Cut the frame or selection from the image, depending on Editor focus. If cutting a selection, this will add it to the brushes.
  • Copy Frame / Selection - Copy the frame or the selection from the image, depending on Editor focus. If copying a selection, this will add it to the brushes.
  • Paste Frame / Selection - Paste the frame from the clipboard, or paste the selection from the clipboard to the image, depending on the current focus of the IDE. When pasting into an image, you should note that this option is designed only for use with clipboard data from external programs. When you cut or copy to the clipboard from an image in the Image Editor, this is added to the brushes in the top right of the editor, and you should select a brush to draw with the cut/copied selection rather than try to paste it into the image. Also note that a number of programs will have issues pasting transparencies from the clipboard into GameMaker Studio 2 image editor due to the format used by the clipboard.
  • Select All - Select the full image.
  • Cancel Selection - Clear any selection from the image.
  • Invert Selection - Invert the current selection.
  • Add Frame - Add a new frame to the sprite.
  • Insert Frame - Insert a new frame after the one currently showing.
  • Delete Selected Fames - Remove the selected frame from the sprite (you can use / + and/or + to select multiple frames).
  • Import Image(s) - Load an image to be used as a frame from disk. Note that you can load multiple frames at once by selecting multiple individual files to import in the explorer window. Once you have selected your file or files you will be shown the following Import window:
    Here you can choose how the imported image should be added to the current sprite, permitting you to resize the entire draw canvas, crop or pad it, and also set the "anchor" position for placing the new image(s).

    Since this is a single image and we want to convert it into individual frames, we can use this option which will then open the following window:
    Here you can set how the single image should be split to create the frames, setting the number of frames, the width and height of the frames and any including any offset values.
  • Import Strip Image - This option will open a file explorer and prompt you to look for a single strip image file to be used to create frames from. Once selected an image the Import strip window will open, exactly the same as for the Convert To Frames option shown above.
  • Export to PNG - Selecting this option will export the sprite as a single PNG image. If the sprite has more than a single image, then a "strip" sprite will be created with all the images, one after another, and the save file name will automatically be appended with "_stripX", where "X" is the number of frames exported. For example, a five frame sprite will be exported as "sprite_name_strip5.png".
  • Resize All Frames - When you select this option it will open the following window where you can set the way that you want to resize all the animation frames that comprise the sprite:
    You can choose to either Scale the image, or to Resize the canvas. If you choose to scale the image then you simply select the new width and/or height for each frame, whether to scale using the same aspect ratio, and also whether to scale in pixels or percent. Note that when scaling you can enable linear interpolation on or off, where having it on will cause the pixels to blur or "smear" as the image is scaled (in general, for hi resolution art you want this on, but for pixel art you want this off).

    The option to resize the canvas will not alter the pixels within the image but instead increase or decrease the size of the canvas that the images are being drawn on. You can set where the "anchor" for the resize should be using the arrow icons, then set the new size (in pixels or as a percentage) as well as set whether to maintain the aspect ratio of the original frames. Note that making the frame smaller than the original sprite images will crop the pixels.
  • Crop All Frames To Selection - This option will only be available when you have an area of the image selected in the main draw canvas. When you then choose this option, the image being edited (and all other frames in the animation) will be cropped to the boundaries of the selection.
  • Auto Trim All Frames - This option permits you to trim all the zero-alpha pixels from around the edges of a sprite. This will take into consideration all image frames within the sprite, so each side will crop to the edge of whichever frame has the furthest non-zero alpha pixel.
  • Reverse Frames - Selecting this will simply reverse the frame order for the sprite.
  • Mirror - This option permits you to mirror either the current frame all all the frames that comprise the sprite along the vertical axis.
  • Flip - This option permits you to flip either the current frame all all the frames that comprise the sprite along the horizontal axis.
  • Rotate all Frames (clockwise 90°) - This option will rotate all the frames of the animation 90° clockwise.
  • Rotate all Frames (anti-clockwise 90°) - This option will rotate all the frames of the animation 90° anti-clockwise.

 

ViewView

The View menu has the following options:

  • 1:1 - Scale the image to be 1:1 with the resolution (ie: actual size) and center it in the image edit window.
  • Fit Screen - Scale the image to fill the image edit window.
  • Show grid - Toggle the pixel grid. This is the same as the grid button found in the Canvas Controls (explained further up this page).
  • Onion Skin Settings - Change the onion skin settings. This will open the following window where you can set the frames shown by the onion skin button, as well as the opacity and colour of the skinned frames.

 

EffectsEffects

The Effects menu is where you can add your own custom effects plugins, and has the following effects by default:

  • Blur - Apply a blur effect to the image. You can set the intensity as well as the type of blur to use (box or gaussian) and set whether to only blur along the horizontal or vertical axis, or along both. You can then choose to apply this to only the current layer, visible layers or all layers.
  • Greyscale - Desaturate the image by a determined amount. You can choose to apply this to only the current layer, visible layers or all layers.
