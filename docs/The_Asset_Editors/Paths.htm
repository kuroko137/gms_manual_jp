パスエディタ

ゲーム作りを進めていると、そのうちインスタンスをレベル上のパスに追従させたいことがあるかもしれません。これはルームの位置データを入れた配列を作成し、その位置データ間をインスタンスに移動させるコードを書くことで実現できますが、位置データの調整が難しく、インスタンスごとにテストして調整しなくてはならないため非常に手間がかかります。そうした理由から、GameMaker Studio 2 にはパスリソースと専用のパスエディタが用意されています。

基本的な扱い方はとてもシンプルで、パスエディタを開いてパスを書き、オブジェクトの Creation イベントにアクション（またはコード）を追加してそのパスに追従させるといったものです。さらにはパスに追従する速度を変えたり、そのほかにもルーム内におけるパスの位置、向きに関するアクションが複数あります。パスリソースを作成すると、まずパスエディタのウィンドウが開き、次のオプションが表示されます。

 

パスのプロパティパスのプロパティ

GameMaker Studio 2 の他のリソースと同じように、新しく作成したパスには制作中のゲームで識別できるよう、固有の名前をつけなくてはなりません。よく使われているのは、接頭語、および接尾語を入れる命名方法で（例: path_Enemy_1、PowerUp_path など）、こうすればコードやアクションから呼び出すときも簡単に見分けることができます。

名前をつけたら、右側のメインエディタにポイントを置いてパスの定義をしましょう。それぞれのポイントはリストに追加され、位置と速度が表示されます。パスのポイント速度値は、インスタンスがパスに沿って移動する速度をパーセント単位で決めるものです（デフォルトでは 100%）。たとえばインスタンスのパスに沿う速度が 2 （ステップ毎に 2 ピクセル）で始まったとして、速度値が 50%になっているポイントがある場合、インスタンスの速度はこのポイントに近づくにつれ 1 （パスの速度の 50%）に減少することになります。同じように速度が 150%であれば、ポイントに近づくごとにインスタンスが加速します。このように、コードを書かなくてもダイナミックな動きを作ることが可能です。

覚えておきたい点として、リストに並んでいる定義済みポイントを素早くダブルクリックするとエディタが選択したポイントまで移動し、ゆっくりダブルクリックすると値 （X, Y, 速度）を直接編集することができます。

また、パス全体の接続タイプ、パスを閉じるか閉じないか（閉じられたパスは最初と最後のポイントをつなぎ合わせます）を指定することもできます。接続タイプは直線またはスムースに設定できます。直線接続はパスのポイントを直線的につなぎ合わせるだけで、ポイント間で角度が変化します。スムース接続はパスをなだらかにし、ポイントの上を通過しない近似線となる場合が多くなりますが、スムース補間による自然なカーブを形成することができます。このカーブは 1 から 8 までの精度を指定可能で（デフォルトでは 4）、8 にすると最もなだらかになります。次の画像は、これらのオプションがパスにどう影響するかのサンプルです。


パスの編集パスの編集

編集ウィンドウをクリックするとポイントを追加でき、クリックを続けるとさらにポイントを追加できます。エディタで置いたポイントは、ドラッグして移動させたり、左側に表示されている X 座標、Y 座標、速度を変更することでも移動できます（上にあるパスのプロパティを参照）。 / を押しながら複数のポイントを左クリックすれば同時に選択でき、ポイントをまとめて好きな位置までドラッグすることができます。を押すと、選択中のポイントを削除できます。

エディタのウィンドウを右クリックすると、次の追加メニューが開きます。

メニューの各項目にはこのような機能があります。

  • ポイントを削除 - 選択中のポイントを削除します。
  • パスにフォーカス - エディタのビューをパスの中心地点に移動させます。
  • 逆順 - 各ポイントの順番を逆にします。これはポイントを移動させるのではなく、進行順を変更します。つまり先頭のポイントが末尾のポイントになるということです。
  • 上下反転 - パスを縦軸に沿って反転させます。
  • 左右反転 - パスを横軸に沿って反転させます。

 

 

ツールボックスツールボックス

上部にあるのはツールボックスボタンで、エディタのウィンドウの表示形式を操作することができます。背景のグリッドをオン / オフにしたり、次のように追加したポイントをグリッドにスナップさせるかどうかを設定できます。

The snap values indicate the size of the grid cells and this can be changed to any value you require. The rest of the tools are dedicated to zooming in and out and panning the editor area, and you can click the button to reset to be 1:1 with the display. Note that if you pan too far from where you want to draw the path, you can use the right mouse button menu, and then click Pan to Path to scroll back to where you were working.


Path PointsPath Points

Paths are simply a collection of points joined together by a line. In the editor, you place these points by using the left mouse button . You can add points anywhere in the room editor and they will be added to the point list consecutively to make a path. If you place a point near a connecting line or another node, the new point will be inserted in the path rather than added to it (you can set the detection distance for this in the Preferences). You can press / and click on already placed points to select them, and then any operations done on the path points like moving them, will be applied to all the selected points as a group. To remove a point or selection of points you can right-click and select "Delete Point" or press the key.

Note that you can also use the following shortcuts to perform some extra operations that are not available through the right mouse button menu on the choose path point(s):

  • T + : Translate (move) the path points
  • R + : Rotate the path points
  • S + : Scale the path points
