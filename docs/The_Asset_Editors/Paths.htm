パスエディタ

ゲーム作りを進めているうちに、インスタンスをパスに追従させたい時があるかもしれません。こうした動きは、ルームの位置情報を入れた配列を作り、インスタンスにその位置まで移動させるコードを書くことで実現できますが、位置の設定が難しいうえに、インスタンスごとに調整を加える必要があるため大変になってきます。そのような理由から、GameMaker Studio 2 にはパスリソース、および専用のパスエディタが用意されています。

基本的な使い方はとてもシンプルで、パスエディタを開いてパスを描き、オブジェクトの Creation イベントにアクション（またはコード）を追加して特定のパスに追従させるだけです。また、パスに追従する速度を設定したり、そのほかにもルームに置かれたパスの位置、向きを設定できるアクションがいくつかあります。パスリソースが作成されると、まずパスエディタのウィンドウが開き、次のオプションが表示されます。

 

パスのプロパティパスのプロパティ

GameMaker Studio 2 の他のリソースと同じように、パスを作成したら制作中のゲームで区別のつくような固有の名前をつけなくてはなりません。よく使われている命名法は、"_" で区切られた接頭語、および接尾語を入れるというもので（例: path_Enemy_1, PowerUp_path ）、このように命名すれば、コードやアクションから呼び出すときも簡単に見分けることができます。

名前をつけたら、右側のメインエディタからポイントを置き、パスを形作りましょう。置かれたポイントはリストに追加され、位置と速度が表示されます。ポイントの速度値は、インスタンスがパスに沿って移動する速度をパーセント単位で決めるものです（デフォルトでは 100%）。たとえばインスタンスのパスに沿う速度が 2 （ステップ毎に 2 ピクセル）で始まったとして、速度値が 50%になっているポイントがある場合、インスタンスの速度はこのポイントに近づくにつれ 1 （パスの速度の 50%）へと減少することになります。同じように速度が 150%であれば、ポイントに近づくにつれインスタンスが加速します。このようにコードを書かなくても、ダイナミックな動きを作ることが可能です。

覚えておきたい点として、リストに追加されたポイントを素早くダブルクリックするとビューをポイントの位置まで移動でき、ゆっくりダブルクリックすると値 （ X, Y, 速度）を直接編集できます。

さらに、パスの接続タイプと、パスを閉じるかどうかを指定可能です。パスを閉じると、最初と最後のポイントがつなぎ合わされます。接続タイプは直線またはスムースから選ぶことができ、直線接続にした場合はパスのポイントが直線的につなぎ合わされ、ポイントの位置で角度が変化します。スムース接続はパスをなだらかにするため、ポイントの位置を通らない近似線になりがちですが、スムース補間による自然なカーブ（曲線）を形成することができます。このカーブは 1 から 8 までの精度を指定可能で（デフォルトでは 4）、8 にすると最もなだらかになります。これらのオプションを設定することで、パスがどのように変化するかを示している例が次の画像です。


パスの編集パスの編集

ポイントは編集ウィンドウをクリックして追加します。繰り返しクリックすればどんどん追加可能です。エディタから置いたポイントはドラッグをするか、左側に表示されている X 座標、Y 座標、速度を変えることで移動できます（パスのプロパティを参照）。 / を押しながら複数のポイントを左クリックすると同時選択が可能で、ポイントをまとめて好きな位置までドラッグできます。を押すと、選択しているポイントを削除できます。

エディタのウィンドウを右クリックした場合は、次のサブメニューが開きます。

メニューの各項目には次の機能があります。

  • ポイントを削除 - 選択中のポイントを削除します。
  • パスにフォーカス - エディタのビューをパスの中心地点に移動させます。
  • 逆順 - 各ポイントの順番を逆にします。これは位置を動かすのではなく、ポイントの進行順を変更します。つまり最初のポイントが最後のポイントになるということです。
  • 上下反転 - パスを縦軸に沿って反転させます。
  • 左右反転 - パスを横軸に沿って反転させます。

 

 

ツールボックスツールボックス

上に並んでいるのはツールボックスボタンで、エディタのウィンドウ表示方法をコントロールできます。背景のグリッド表示を切り替えたり、置いたポイントをグリッドにスナップさせるかどうかを設定可能です。

グリッドのセルサイズはスナップの値でもあり、好きな値に変更可能です。それ以外はズームイン、ズームアウト、フォーカス用のツールとなり、ボタンをクリックするとエディタを等倍表示にリセットできます。また、パスを描いている場所から離れすぎてしまった場合、右クリックメニューを開いてパスにフォーカスをクリックすると、元の場所まで移動できます。


パスのポイントパスのポイント

パスとはすなわち、線でつなぎ合わされたポイントが集まったものです。エディタでは、ポイントを左クリックで配置できます。ポイントはエディタのどこにでも配置でき、リストに次々と追加されてパスとなります。線、または他のノードの近くにポイントを追加した場合、ポイントはその位置に新しく挿入されます（環境設定から挿入されるまでの距離を指定できます）。配置したポイントは / を押しながらクリックすると自由に選択可能で、そのまま移動などの操作を行なった場合、選択したポイントすべてに反映されます。ポイントを削除するには、右クリックして「ポイントを削除」を選ぶか、キーを押してください。

また次のショートカットを利用することで、ポイントの右クリックメニューからは実行できない特別な操作ができます。

  • T + : パスの全ポイントを移動
  • R + : パスの全ポイントを回転
  • S + : パスの全ポイントを拡大縮小
