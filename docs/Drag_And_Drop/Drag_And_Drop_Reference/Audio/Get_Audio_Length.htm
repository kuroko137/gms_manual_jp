Get Audio Length

このアクションはゲームに存在するサウンドの長さを取得します。長さを取得するサウンドリソース、値を返す対象変数を指定してください（フラグを追加して一時変数にすると、イベント / スクリプトが終了するまで利用できる変数がアクションによって作成されます）。長さは秒単位で対象変数（一時変数でも可）に返され、値は 10 進数となります。

 

アクション構文:

引数:

引数 説明
サウンド 長さを取得するサウンドリソース
対象 戻り値を格納する対象変数

 

例:

上記のアクションブロックコードは、指定したサウンドリソースの再生時間を取得し、一時的ローカル変数に格納します。さらに 0 からサウンドの長さの間でランダムな値を生成し、新しい一時的ローカル変数に保存します。生成されたランダムな値をサウンドの再生開始位置にセットし、最初に指定したサウンドを再生します。
