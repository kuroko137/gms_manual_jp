Play Audio

このアクションは指定されたサウンドを再生状態にします。アセットエクスプローラーからサウンドを選択し、"ループ" フラグでサウンドをループさせるかを設定します（ループ中のサウンドは停止されるまで再生を繰り返します）。

 

アクション構文:

引数:

引数 説明
サウンド 再生するサウンドリソース
ループ サウンドをループ再生するかどうか（デフォルトではオフ）

 

用例:

上記のアクションブロックコードは、左マウスボタンが押されたかをチェックし、押された場合はグローバル変数が true かどうかをチェックします。true でない場合、サウンドを再生（ループ）してグローバル変数を true に設定します。true であった場合、サウンドを再生停止し、グローバル変数を false に設定します。
