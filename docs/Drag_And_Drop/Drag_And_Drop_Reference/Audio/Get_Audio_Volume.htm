Get Audio Volume

このアクションは指定されたサウンドの音量を返します。アセットエクスプローラーからサウンドを選び、音量の値を返す対象変数い（ローカル一時変数に切り替え可）を指定してください。返り値は 0 から 1 の間で、0 はミュート、1 はフル音量となります。

 

アクション構文:

引数:

引数 説明
サウンド ボリュームを取得するサウンドリソース
対象 ボリュームの返り値を格納する変数

 

例:

The above action block code checks a global variable to see if it is true or false. If it is true then the volume of the given sound is retrieved and stored in a temporary local variable. The value is then checked to see if it is not equal to 0.5, and if it is not, then the volume of the sound is set to 0.5. If the global variable evaluates to false, then the sound has its volume set to 1.
