If End Of Buffer

This action can be used to check if the read/write position has reached the end of the buffer. You supply the unique buffer ID value as returned by the action Create Buffer and then the action will return true if the read/write position has reached the end of all the available data in the buffer, or false if it has not. You can flag "not" to make the action check if the read/write position is not at the end of the buffer.
