Random Action Library

The Random library has the following actions for dealing with random number generation and chance operations:

Get Random Number
Randomise
Choose

 

 

 

 

 

It is worth noting that GameMaker Studio 2 will use the same seed when generating random numbers each time you run the game from the IDE. This is to facilitate debugging and means that you will get the same initial results when generating random values, but this will not occur when you have compiled the game into an executable (a new seed will be generated each time the game is run). If you do not wish this behaviour when testing, then you should call the Randomise action once at the start of the game, before generating any random values.
