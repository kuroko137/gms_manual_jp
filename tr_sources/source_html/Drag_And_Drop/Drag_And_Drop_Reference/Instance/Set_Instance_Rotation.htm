{WITH_ENG} Set Instance Rotation

This action block sets the image_angle (rotation) of the instance. The angle is measured in degrees, with the right being 0°, up being 90°, left being 180° and down being 270°. Set this variable to 0 to reset the instance, meaning that it will draw any sprite assigned to as it was defined in the sprite editor. Please note that for changes in this action to be visible, the instance should have either no draw event (and so GameMaker Studio 2 will default draw the sprite) or be drawn using Draw Self action.

 

Action Syntax:

Arguments:

 

Argument Description
Angle The new angle to use (from 0 to 360).
